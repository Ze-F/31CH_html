通用工具箱 (反复用到)

📐 Hubbard 的 Fourier 约定: $\widehat{f}(\xi) = \int_{\R^n} f(x) e^{2\pi i x\cdot\xi}\,dx$. 注意是 $+2\pi i$ 不是 $-2\pi i$. 这影响:
  • 微分律: $\widehat{f'}(\xi) = -2\pi i\xi\,\widehat{f}(\xi)$, $\widehat{f''}(\xi) = -4\pi^2\xi^2\,\widehat{f}(\xi)$.
  • 张量积律: 矩形 indicator 的 FT = 1D indicator FT 的乘积 (Fubini).
  • 卷积定理: $\widehat{f * g} = \widehat{f}\cdot\widehat{g}$.
⚡ Laplace 的导数律: $\LT(y')(s) = sY(s) - y(0)$, $\LT(y'')(s) = s^2 Y - sy(0) - y'(0)$.
反 Laplace 利器: 部分分式 + 卷积定理 $\LT(f * g) = \LT(f)\LT(g)$.

§2 Part I — Fourier 变换 (4 题)

Part I 四道题互不相同, 各自训练一个 FT 的核心技巧. 顺序由易到难:

  1. I.1 — 对称区间 indicator → sinc: 用 $e^{ia} - e^{-ia} = 2i\sin a$.
  2. I.2 — 矩形 indicator → 张量积: Fubini 把 2D 拆成两个 1D.
  3. I.3 — 双侧指数 → Lorentzian: 在 0 处切, 改变量.
  4. I.4 — Gaussian self-dual: 用 differentiation under integral (DUI) + IBP 得到一阶 ODE in $\xi$.

§2.1 Problem I.1 — 对称区间 indicator → sinc

Problem I.1. 计算 $f(x) = \1_{[-a,a]}(x)$ 的 Fourier 变换 ($a>0$).

策略. 直接代入定义, 在 $\xi=0$ 与 $\xi\ne 0$ 分情况处理. 关键技巧是把 $e^{2\pi i a\xi} - e^{-2\pi i a\xi}$ 识别为 $2i\sin(2\pi a\xi)$ — 这是所有"对称区间"问题的核心代数动作.

展开计算
$\widehat{f}(\xi) = \int_{-a}^{a} e^{2\pi i x\xi}\,dx$. 当 $\xi\ne 0$: $$ \widehat{f}(\xi) = \left[\frac{e^{2\pi i x\xi}}{2\pi i\xi}\right]_{-a}^{a} = \frac{e^{2\pi i a\xi} - e^{-2\pi i a\xi}}{2\pi i\xi} = \frac{2i\sin(2\pi a\xi)}{2\pi i\xi} = \frac{\sin(2\pi a\xi)}{\pi\xi}. $$ $\xi=0$ 处 $\widehat{f}(0) = 2a$, 与 $\lim_{\xi\to 0}\sin(2\pi a\xi)/(\pi\xi) = 2a$ 一致. 写成统一形式: $$\boxed{\,\widehat{f}(\xi) = \frac{\sin(2\pi a\xi)}{\pi\xi} = 2a\,\sinc(2a\xi)\,}$$ 其中 $\sinc(x) = \sin(\pi x)/(\pi x)$.
反思. 这是"不光滑 → 慢衰减"的范例: $f$ 在 $\pm a$ 处只有跳跃间断, 所以 $\widehat{f}$ 只衰减如 $1/|\xi|$. 如果 $f$ 是 $C^k$ 光滑, $\widehat{f}$ 会衰减到 $1/|\xi|^{k+1}$ 阶. 这是 FT 的"光滑性 ↔ 衰减性"对偶, 后面 I.4 的 Gauss 是极致 (无穷光滑 ↔ 指数衰减).

§2.2 Problem I.2 — 矩形 indicator → 张量积 (用 Fubini)

Problem I.2. 计算 $f(x,y) = \1_{[-a,a]\times[-b,b]}(x,y)$ 的 FT.

策略. 不要再做一遍 2D 积分! 矩形 indicator 张量积分解 $\1_{[-a,a]\times[-b,b]}(x,y) = \1_{[-a,a]}(x)\cdot\1_{[-b,b]}(y)$, 而 $e^{2\pi i(x\xi_1 + y\xi_2)} = e^{2\pi i x\xi_1}\cdot e^{2\pi i y\xi_2}$. Fubini 直接把积分劈成两个 I.1 的副本.

展开计算
$$ \widehat{f}(\xi_1,\xi_2) = \int_{\R^2} \1_{[-a,a]}(x)\1_{[-b,b]}(y) e^{2\pi i(x\xi_1 + y\xi_2)}\,dx\,dy = \left(\int_{-a}^a e^{2\pi i x\xi_1}dx\right)\left(\int_{-b}^b e^{2\pi i y\xi_2}dy\right). $$ 每个因子用 I.1: $$\boxed{\,\widehat{f}(\xi_1, \xi_2) = \frac{\sin(2\pi a\xi_1)}{\pi\xi_1}\cdot\frac{\sin(2\pi b\xi_2)}{\pi\xi_2}\,}$$
反思. 这是 FT 的张量积律: 若 $h(x,y) = f(x)g(y)$, 则 $\widehat{h}(\xi_1,\xi_2) = \widehat{f}(\xi_1)\widehat{g}(\xi_2)$. 推广到 $n$ 维 box 一样: $\widehat{\1_{[-a_1,a_1]\times\cdots\times[-a_n,a_n]}}(\xi) = \prod_{j=1}^n \frac{\sin(2\pi a_j\xi_j)}{\pi\xi_j}$. 这是多维 FT 经常用 Fubini 拆成 1D的根本.

§2.3 Problem I.3 — 双侧指数 → Lorentzian

Problem I.3. 计算 $f(x) = e^{-a|x|}$ 的 FT ($a>0$).

策略. 绝对值不可微, 但有分段处理: 在 $x=0$ 切开, $x<0$ 时 $|x|=-x$, $x>0$ 时 $|x|=x$. 在负半段做变量代换 $y=-x$ 把积分翻到正半段 (但带个 $e^{-2\pi i y\xi}$ 而不是 $e^{2\pi i y\xi}$). 两个积分加起来用公分母, 实部相加, 虚部相消, 得到 Lorentzian.

展开计算
$$ \widehat{f}(\xi) = \int_{-\infty}^0 e^{ax}e^{2\pi i x\xi}\,dx + \int_0^{\infty} e^{-ax}e^{2\pi i x\xi}\,dx. $$ 第一项做 $y = -x$: $$\int_{-\infty}^0 e^{ax}e^{2\pi i x\xi}\,dx = \int_0^{\infty} e^{-ay}e^{-2\pi i y\xi}\,dy = \frac{1}{a + 2\pi i\xi}.$$ 第二项直接积: $$\int_0^{\infty} e^{-(a - 2\pi i\xi)x}\,dx = \frac{1}{a - 2\pi i\xi}.$$ 相加: $$\widehat{f}(\xi) = \frac{1}{a + 2\pi i\xi} + \frac{1}{a - 2\pi i\xi} = \frac{2a}{a^2 + 4\pi^2\xi^2}.$$ $$\boxed{\,\widehat{e^{-a|x|}}(\xi) = \frac{2a}{a^2 + 4\pi^2\xi^2}\,}$$ 特别地, $a=1$ 给 $\widehat{e^{-|x|}}(\xi) = 2/(1+4\pi^2\xi^2)$, 正是 III.2 题用到的提示.
反思. "指数衰减 → 代数衰减" 的对偶: 空间侧指数衰减 ($e^{-a|x|}$), 频率侧只代数衰减 ($1/(1+\xi^2)$). 反过来 (FT 是自伴的, 取反向变换得 $\widehat{1/(a^2+4\pi^2\xi^2)} \propto e^{-a|x|}$) 也成立: 代数衰减 → 指数衰减. 而 Gauss 是双侧指数, 极端对偶 (I.4).

§2.4 Problem I.4 — Gaussian 自对偶 (DUI + ODE 法)

Problem I.4. 证明 $f(x) = e^{-\pi x^2}$ 是其自身的 Fourier 变换.

策略. 这是有名的题, 也是 Hubbard 课本的代表. 有三种证明:

  • (推荐) DUI + IBP + ODE 法: 把 $\widehat{f}(\xi)$ 看作 $\xi$ 的函数 $F(\xi)$, differentiate under the integral 得 $F'(\xi)$, IBP 把 $xe^{-\pi x^2}$ 变成 $-\frac{d}{dx}\frac{e^{-\pi x^2}}{2\pi}$, 得到 $F' = -2\pi\xi F$. 这是一阶线性 ODE, 解出 $F = F(0)\cdot e^{-\pi\xi^2}$. 用 Gauss 积分公式 $\int e^{-\pi x^2} dx = 1$ 给 $F(0)=1$.
  • (快但需 contour) 配方法: $-\pi x^2 + 2\pi i x\xi = -\pi(x - i\xi)^2 - \pi\xi^2$, 积分 $\int e^{-\pi(x-i\xi)^2}\,dx$ 用 contour shift 等于 $\int e^{-\pi x^2}\,dx = 1$. 严格性需要复分析的回路定理.
  • (理论方法) Hermite 函数论: $e^{-\pi x^2/2}$ 是 FT 算子的本征函数, 本征值 $+1$ 是因为它是 0 阶 Hermite. 太重武器, 考试不写.

考试推荐 DUI + IBP + ODE 法: 最 rigorous, 最容易记, 一气呵成. 关键技巧: 看见 $xe^{-\pi x^2}$ 立刻想到它是 $-\frac{1}{2\pi}(e^{-\pi x^2})'$, 这样就能做 IBP.

展开计算 (DUI 法)
令 $F(\xi) = \widehat{f}(\xi) = \int_{\R} e^{-\pi x^2} e^{2\pi i x\xi}\,dx$. 用 DCT (积分核与 $\xi$ 偏导都被 $|x|e^{-\pi x^2}$ 控制) 把 $\partial_\xi$ 与积分交换: $$F'(\xi) = \int_{\R} 2\pi i x e^{-\pi x^2} e^{2\pi i x\xi}\,dx.$$ 关键观察: $\frac{d}{dx} e^{-\pi x^2} = -2\pi x e^{-\pi x^2}$, 所以 $2\pi i x e^{-\pi x^2} = -i\frac{d}{dx}e^{-\pi x^2}$. IBP: $$F'(\xi) = -i\left[e^{-\pi x^2} e^{2\pi i x\xi}\right]_{-\infty}^{\infty} + i\int_{\R} e^{-\pi x^2}\cdot 2\pi i\xi\cdot e^{2\pi i x\xi}\,dx = 0 - 2\pi\xi F(\xi).$$ 这是一阶线性 ODE: $\frac{d}{d\xi}\log F = -2\pi\xi$, 积分得 $$F(\xi) = F(0) e^{-\pi\xi^2}.$$ $F(0) = \int_\R e^{-\pi x^2}\,dx = 1$ (经典 Gauss 积分, 用极坐标计算 $\left(\int_\R e^{-\pi x^2}dx\right)^2 = \int_{\R^2} e^{-\pi r^2}\,r\,dr\,d\theta\cdot 2\pi$ 得到). 因此 $$\boxed{\,\widehat{e^{-\pi x^2}}(\xi) = e^{-\pi\xi^2}\,}$$
反思. $e^{-\pi x^2}$ 的归一化是 Hubbard 约定下"自然"的: 指数里是 $-\pi x^2$ 而不是 $-x^2/2$. 这恰好让 FT self-dual. 如果你换约定为 $e^{-x^2/2}$, 那它在 Hubbard 约定下 FT 是 $\sqrt{2\pi}\cdot e^{-2\pi^2\xi^2}$, 不是自对偶. 记忆口诀: "Hubbard 约定下, $e^{-\pi x^2}$ 是 FT 的不动点, 也是 Hermite 函数序列的零阶元."

II.1. $\LT(1)(s) = \int_0^\infty e^{-st}\,dt = [-e^{-st}/s]_0^\infty = 1/s$ ($s > 0$).

记忆口诀.

  • "$\sin / \cos$ → 分母都是 $s^2 + b^2$, 分子是 $b$ 还是 $s$ 看是 sin 还是 cos" — 简记 "$\sin$ 给 $b$, $\cos$ 给 $s$".
  • "$t^n$ → $n!/s^{n+1}$" — 记 $t^n$ 多了一个 $t$, $s$ 就多一阶, 且带 factorial.

§4 Part III — PDE/ODE 应用 (4 题)

这是 Practice Final 最重要的一部分: Part I/II 是工具, Part III 教你怎么用工具解 PDE/ODE. 四道题分两组:

  • III.1, III.2: FT 解 PDE — "在 $x$ 维做 FT, 把空间导数变成 $\xi$ 倍数, 留下 $t$ 维 ODE / 完全代数方程, 再做反 FT 识别成 convolution."
  • III.3, III.4: LT 解 ODE — "把 $y, y', y''$ 都翻译成 $s$ 的代数式, 解 $Y(s)$ 的代数方程, 用部分分式 + 卷积定理反演."

§4.1 Problem III.1 — 热方程 $u_t = u_{xx}$, $u(x,0)=f(x)$

Problem III.1. 用 FT (对 $x$) 解 $u_t = u_{xx}$ on $\R\times(0,\infty)$, 初值 $u(x,0) = f(x)$.

策略 (3 步).

  1. 变换: 对 $x$ 做 FT. 空间二阶导 $u_{xx}$ 变成 $-4\pi^2\xi^2\widehat{u}$ (因为 $\widehat{f''} = (-2\pi i\xi)^2\widehat{f}$). 时间导 $u_t$ 与 FT 交换 (DCT), 仍是 $\partial_t \widehat{u}$.
  2. 解参数 ODE: 对每个固定的 $\xi$, 这是一阶 ODE in $t$: $\partial_t\widehat{u} = -4\pi^2\xi^2\widehat{u}$, $\widehat{u}(\xi,0) = \widehat{f}(\xi)$. 解出 $\widehat{u}(\xi,t) = \widehat{f}(\xi)\,e^{-4\pi^2\xi^2 t}$.
  3. 反演: 识别 $e^{-4\pi^2 t\xi^2}$ 为热核 (heat kernel) $G_t(x) = \frac{1}{\sqrt{4\pi t}}e^{-x^2/(4t)}$ 的 FT (用 Part I.4 + rescaling). 卷积定理: $\widehat{u} = \widehat{f}\cdot\widehat{G_t} = \widehat{G_t * f}$, 所以 $u = G_t * f$.
展开计算 (含 $\widehat{G_t}$ 验证)
Step 1 (FT). 对 $x$ 做 FT, 用微分律 $\widehat{f''}(\xi) = -4\pi^2\xi^2 \widehat{f}(\xi)$: $$\partial_t\widehat{u}(\xi,t) = -4\pi^2\xi^2\widehat{u}(\xi,t), \quad \widehat{u}(\xi,0) = \widehat{f}(\xi).$$ Step 2 (ODE). 一阶线性 ODE in $t$, 解为 $$\widehat{u}(\xi,t) = \widehat{f}(\xi)\,e^{-4\pi^2\xi^2 t}.$$ Step 3 (识别热核 FT). 设 $G_t(x) = (4\pi t)^{-1/2} e^{-x^2/(4t)}$. 用 rescaling $\widehat{g(a\cdot)}(\xi) = a^{-1}\widehat{g}(\xi/a)$ 与 I.4: 写 $G_t(x) = (4\pi t)^{-1/2}e^{-\pi(ax)^2}$ where $a = 1/\sqrt{4\pi t}$, 算得 $$\widehat{G_t}(\xi) = (4\pi t)^{-1/2}\cdot a^{-1}\cdot e^{-\pi(\xi/a)^2} = e^{-4\pi^2 t\xi^2}.$$ Step 4 (卷积定理). $\widehat{u} = \widehat{f}\cdot\widehat{G_t} = \widehat{G_t * f}$. 由 FT 的单射性, $$\boxed{\,u(x,t) = (G_t * f)(x) = \frac{1}{\sqrt{4\pi t}}\int_{\R} e^{-(x-y)^2/(4t)}\,f(y)\,dy\,}$$
反思 (考试要点).
  • "FT 解 PDE 的范式"是这个: 空间 FT → 留下 $t$-ODE → 解 ODE → 反 FT 识别为 convolution.
  • 热核 $G_t$ 是这个 PDE 的基本解 (fundamental solution): $G_t \to \delta_0$ as $t\to 0^+$.
  • $G_t$ 把 $f$ 在尺度 $\sqrt{t}$ 上抹平: 越大的 $t$, smoothing 越强.
  • 考场技巧: 不要去硬算 $\widehat{G_t}$, 直接看 $e^{-4\pi^2 t \xi^2}$ 形式上就是 Gauss, 与 I.4 类比立刻知道是某个 Gauss 的 FT.

§4.2 Problem III.2 — $-u'' + u = f$ (Helmholtz-like)

Problem III.2. 用 FT 解 $-u''(x) + u(x) = f(x)$ on $\R$. 提示: $\widehat{e^{-|x|}}(\xi) = 2/(1+4\pi^2\xi^2)$.

策略. 比 III.1 更直接 — 没有 $t$ 维, 只是纯代数方程. 做 FT 后: $-(-4\pi^2\xi^2)\widehat{u} + \widehat{u} = \widehat{f}$, 即 $(4\pi^2\xi^2 + 1)\widehat{u} = \widehat{f}$. 解出 $\widehat{u}(\xi) = \widehat{f}(\xi)/(1+4\pi^2\xi^2)$, 然后识别 $1/(1+4\pi^2\xi^2)$ 为 $\frac{1}{2}\widehat{e^{-|x|}}(\xi)$ (用提示), 卷积定理给 $u = \frac{1}{2}e^{-|x|} * f$.

展开计算
做 FT: $\widehat{u''} = -4\pi^2\xi^2\widehat{u}$. 因此 $$-\widehat{u''}(\xi) + \widehat{u}(\xi) = \widehat{f}(\xi) \implies (4\pi^2\xi^2 + 1)\widehat{u}(\xi) = \widehat{f}(\xi) \implies \widehat{u}(\xi) = \frac{\widehat{f}(\xi)}{1 + 4\pi^2\xi^2}.$$ 由提示, $\frac{1}{1+4\pi^2\xi^2} = \frac{1}{2}\widehat{e^{-|x|}}(\xi)$, 所以 $\widehat{u} = \frac{1}{2}\widehat{e^{-|\cdot|}}\cdot\widehat{f} = \widehat{\frac{1}{2}e^{-|\cdot|} * f}$. 因此 $$\boxed{\,u(x) = \frac{1}{2}(e^{-|\cdot|} * f)(x) = \frac{1}{2}\int_\R e^{-|x-y|}f(y)\,dy\,}$$
反思. 核 $K(x) = \frac{1}{2}e^{-|x|}$ 是算子 $L = -\partial_x^2 + 1$ 的Green 函数: $LK = \delta_0$. 这是常系数 ODE/PDE on $\R$ 的标配解法 — FT 把微分算子变代数除法, 反演把代数除法变 convolution. 推广: 同样方法解 $(-\Delta + m^2)u = f$ on $\R^n$ 得 Bessel-type Green 函数.

§4.3 Problem III.3 — $y' + 3y = e^{-t}$, $y(0)=2$ (LT 解一阶 ODE)

Problem III.3. 用 Laplace 变换解 $y'(t) + 3y(t) = e^{-t}$, $y(0)=2$.

策略 (3 步).

  1. 变换: 用 $\LT(y')(s) = sY(s) - y(0)$ 把 ODE 变代数方程.
  2. 解 $Y(s)$: 一行代数得到 $Y(s) = (2s+3)/[(s+1)(s+3)]$.
  3. 反演 (部分分式): 分解 $Y(s) = A/(s+1) + B/(s+3)$, 查表反演.

关键技巧: 部分分式时用 "代入根 → 得系数" 法 (cover-up method): 把 $s = -1$ 代入 $2s+3 = A(s+3) + B(s+1)$ 得 $1 = 2A$, $A = 1/2$; $s = -3$ 得 $-3 = -2B$, $B = 3/2$.

展开计算 + 验证
$\LT(y') = sY - 2$, $\LT(e^{-t}) = 1/(s+1)$. ODE 变成 $$sY - 2 + 3Y = \frac{1}{s+1} \implies (s+3)Y = 2 + \frac{1}{s+1} = \frac{2s+3}{s+1} \implies Y(s) = \frac{2s+3}{(s+1)(s+3)}.$$ 部分分式: $A = 1/2$, $B = 3/2$, 因此 $$Y(s) = \frac{1/2}{s+1} + \frac{3/2}{s+3} \implies \boxed{\,y(t) = \tfrac{1}{2}e^{-t} + \tfrac{3}{2}e^{-3t}\,}$$ 验证: $y(0) = 1/2 + 3/2 = 2$ ✓. $y' = -\frac{1}{2}e^{-t} - \frac{9}{2}e^{-3t}$, $y' + 3y = -\frac{1}{2}e^{-t} - \frac{9}{2}e^{-3t} + \frac{3}{2}e^{-t} + \frac{9}{2}e^{-3t} = e^{-t}$ ✓.

§4.4 Problem III.4 — $y'' + y = \sin t$, $y(0)=0$, $y'(0)=1$ (LT + 共振)

Problem III.4. 用 Laplace 变换解 $y''(t) + y(t) = \sin t$, $y(0) = 0$, $y'(0) = 1$.

策略. 这题比 III.3 难一点, 因为强迫频率 $\sin t$ 正好与齐次方程 $y'' + y = 0$ 的本征频率共振 (resonance). 反演时会出现 $1/(s^2+1)^2$ — 这种"分母平方"的反演用卷积定理最干净.

反演 $1/(s^2+1)^2$ 的两种方法:

  • (推荐) 卷积: $1/(s^2+1)^2 = [\LT(\sin t)]^2 = \LT(\sin t * \sin t)$, 算 $(\sin*\sin)(t) = \int_0^t \sin(t-\tau)\sin\tau\,d\tau$ 用 product-to-sum identity.
  • (背) 标准表: $\LT^{-1}\{1/(s^2+1)^2\} = \frac{1}{2}(\sin t - t\cos t)$. 也可用 $\LT(t\sin t) = 2s/(s^2+1)^2$ 等导出.
展开计算 (卷积法)
$\LT(y'') = s^2 Y - sy(0) - y'(0) = s^2 Y - 1$. $\LT(\sin t) = 1/(s^2+1)$. ODE 变成 $$(s^2 Y - 1) + Y = \frac{1}{s^2+1} \implies (s^2+1)Y = 1 + \frac{1}{s^2+1} = \frac{s^2+2}{s^2+1} \implies Y(s) = \frac{s^2+2}{(s^2+1)^2}.$$ 拆分: $Y(s) = \frac{(s^2+1)+1}{(s^2+1)^2} = \frac{1}{s^2+1} + \frac{1}{(s^2+1)^2}$. 第一项反演为 $\sin t$. 第二项用卷积: $$\frac{1}{(s^2+1)^2} = \LT(\sin t)\LT(\sin t) = \LT(\sin t * \sin t).$$ 算卷积 (用 $\sin A\sin B = \frac{1}{2}[\cos(A-B) - \cos(A+B)]$): $$ (\sin*\sin)(t) = \int_0^t \sin(t-\tau)\sin\tau\,d\tau = \frac{1}{2}\int_0^t[\cos(t-2\tau) - \cos t]d\tau = \frac{1}{2}\sin t - \frac{t}{2}\cos t = \tfrac{1}{2}(\sin t - t\cos t). $$ 因此 $$y(t) = \sin t + \tfrac{1}{2}(\sin t - t\cos t) = \boxed{\,\tfrac{3}{2}\sin t - \tfrac{t}{2}\cos t\,}$$ 验证: $y(0)=0$ ✓. $y' = \cos t + \frac{t}{2}\sin t$, $y'(0)=1$ ✓. $y''+y = \sin t$ ✓ (算一下).
反思 (共振). 关键观察: 解里有 $-\frac{t}{2}\cos t$ — 一个线性增长 × 周期函数的"secular term". 这是因为强迫 $\sin t$ 频率恰好等于齐次解的固有频率 (the natural frequency of $y''+y=0$), 造成 amplitude growth. 物理对应: 一个单摆被以其固有频率推动, 振幅线性增长 (直到非线性效应启动). LT 解法自动揪出这个 secular term, 不需要"特解 + 齐次"的人工试解. 这是 LT 最漂亮的应用之一.

§5 Part IV — 平面曲线曲率 (4 题)

Part IV 四题套同一个公式:

$$\kappa(t) = \frac{|x'(t)y''(t) - y'(t)x''(t)|}{((x'(t))^2 + (y'(t))^2)^{3/2}}.$$

分母里 $|c'(t)|^3$ 的 $3/2$ 次方源自 chain rule (非单位速度参数化). 这是 Lec 10.2 给出的"general parametrization formula". 单位速度时 ($|c'|=1$) 公式退化到 $\kappa = |x'y'' - y'x''|$.

策略. 4 道题都是套公式 → 简化, 没有特别技巧. 反而是要读几何意义: 哪里曲率最大、是否有拐点、$\kappa\to 0$ 的渐近行为.

§5.1 IV.1 — 抛物线 $c(t) = (t, t^2)$

展开计算
$x' = 1, y' = 2t, x'' = 0, y'' = 2$. $|x'y'' - y'x''| = 2$. $(x')^2 + (y')^2 = 1 + 4t^2$. $$\boxed{\,\kappa(t) = \frac{2}{(1 + 4t^2)^{3/2}}\,}$$
几何: $\kappa(0) = 2$ (顶点处, osculating circle 半径 $1/2$). $\kappa\to 0$ as $|t|\to\infty$ (抛物线在无穷远处变直).

§5.2 IV.2 — 半径 $a$ 的圆 $c(t) = (a\cos t, a\sin t)$

展开计算
$x' = -a\sin t, y' = a\cos t, x'' = -a\cos t, y'' = -a\sin t$. $|x'y'' - y'x''| = a^2(\sin^2 t + \cos^2 t) = a^2$. $(x')^2 + (y')^2 = a^2$. $$\boxed{\,\kappa(t) = a^2/(a^2)^{3/2} = 1/a\,}$$
几何: 圆周上每一点曲率都是 $1/a$ — 这是"曲率 = osculating circle 半径的倒数"的基准案例. 反过来: 任何平面曲线在曲率为 $\kappa$ 的点的 osculating 圆半径就是 $1/\kappa$.

§5.3 IV.3 — 三次曲线 $c(t) = (t, t^3)$

展开计算
$x' = 1, y' = 3t^2, x'' = 0, y'' = 6t$. $|x'y'' - y'x''| = 6|t|$. $(x')^2 + (y')^2 = 1 + 9t^4$. $$\boxed{\,\kappa(t) = \frac{6|t|}{(1+9t^4)^{3/2}}\,}$$
几何: $\kappa(0) = 0$ — 这是拐点 (inflection point), 三次曲线在原点改变凹凸方向, 所以曲率必为 0. 这是"$\kappa = 0$ ⇔ 拐点"的范例 (对于二次连续可微平面曲线).

§5.4 IV.4 — 指数曲线 $c(t) = (t, e^t)$

展开计算
$x' = 1, y' = e^t, x'' = 0, y'' = e^t$. $|x'y'' - y'x''| = e^t$. $(x')^2 + (y')^2 = 1 + e^{2t}$. $$\boxed{\,\kappa(t) = \frac{e^t}{(1 + e^{2t})^{3/2}}\,}$$
几何: $t\to -\infty$ 时 $\kappa\sim e^t \to 0$ (曲线渐近 $x$ 轴, 几乎是直线). $t\to +\infty$ 时 $\kappa\sim e^t/e^{3t} = e^{-2t}\to 0$ (曲线变成"垂直", 也变直). 最大值在唯一临界点 $t^* = -\frac{1}{2}\ln 2$ (从 $\frac{d}{dt}\log\kappa = 1 - \frac{3}{2}\cdot\frac{2e^{2t}}{1+e^{2t}} = 0$ 解出).

§6 Part V — 空间曲线 κ, τ (4 题)

Part V 4 道题套两个公式:

$$\kappa(t) = \frac{|c'(t)\times c''(t)|}{|c'(t)|^3}, \quad \tau(t) = \frac{\det(c'(t), c''(t), c'''(t))}{|c'(t)\times c''(t)|^2}.$$

关键技巧 (节省时间):

  • $c'\times c''$ 用 $\det\begin{pmatrix}\vec i & \vec j & \vec k\\ \cdot \\ \cdot\end{pmatrix}$ 展开. 经验: 第三列经常有 0, 沿第三列展开能省一半工夫.
  • $\det(c', c'', c''')$ 三阶行列式. 如果有上三角结构 (twisted cubic 就是), 直接对角积. 如果第三列只有一个非零 (helix), 沿第三列展开成 $2\times 2$.
  • 圆柱螺旋线 $c(t) = (R\cos t, R\sin t, ht)$ 是原型: $\kappa = R/(R^2+h^2)$, $\tau = h/(R^2+h^2)$ (常数). 记住这个公式可以做 cross-check.
  • $\tau \equiv 0$ ⟺ 曲线在固定平面内 (这是 V.3 的考点). 平面曲线没有"扭".

§6.1 V.1 — 单位螺旋线 $c(t) = (\cos t, \sin t, t)$

展开计算
$c' = (-\sin t, \cos t, 1), |c'|^2 = 2, |c'|^3 = 2\sqrt 2$.
$c'' = (-\cos t, -\sin t, 0)$.
$c''' = (\sin t, -\cos t, 0)$.
$c'\times c'' = \det\begin{pmatrix}\vec i & \vec j & \vec k\\ -\sin t & \cos t & 1\\ -\cos t & -\sin t & 0\end{pmatrix} = (\sin t, -\cos t, 1)$, $|c'\times c''| = \sqrt{2}$. $$\kappa = \sqrt 2/(2\sqrt 2) = \boxed{1/2}.$$ $\det(c', c'', c''')$: 第三列只有第一行非零 (值 1), 沿第三列展开: $$= 1\cdot\det\begin{pmatrix}-\cos t & -\sin t\\ \sin t & -\cos t\end{pmatrix} = \cos^2 t + \sin^2 t = 1.$$ $|c'\times c''|^2 = 2$, 因此 $$\tau = 1/2 = \boxed{1/2}.$$
几何: 常数 $\kappa$ 与常数 $\tau$ 是螺旋线 (helix) 的特征 (Klingenberg Thm. 1.5.7). 反过来: 一个 regular 空间曲线如果 $\kappa, \tau$ 都常数且非零, 必是圆柱螺旋线 (up to rigid motion).

§6.2 V.2 — 加宽螺旋线 $c(t) = (2\cos t, 2\sin t, t)$

展开计算
$c' = (-2\sin t, 2\cos t, 1), |c'|^2 = 5, |c'|^3 = 5\sqrt 5$.
$c'' = (-2\cos t, -2\sin t, 0)$.
$c''' = (2\sin t, -2\cos t, 0)$.
$c'\times c'' = (2\sin t, -2\cos t, 4)$, $|c'\times c''|^2 = 4\sin^2 t + 4\cos^2 t + 16 = 4 + 16 = 20$, $|c'\times c''| = 2\sqrt 5$. $$\kappa = 2\sqrt 5/(5\sqrt 5) = \boxed{2/5}.$$ $\det(c',c'',c''')$: 第三列只有第一行非零 (1), 展开: $$= 1\cdot\det\begin{pmatrix}-2\cos t & -2\sin t\\ 2\sin t & -2\cos t\end{pmatrix} = 4\cos^2 t + 4\sin^2 t = 4.$$ $|c'\times c''|^2 = 20$, $\tau = 4/20 = \boxed{1/5}$. 用螺旋公式 ($R = 2, h = 1$): $\kappa = 2/(4+1) = 2/5$ ✓, $\tau = 1/(4+1) = 1/5$ ✓.

§6.3 V.3 — 平面曲线 $c(t) = (t, t^2, 0)$

展开计算
$c' = (1, 2t, 0)$, $c'' = (0, 2, 0)$, $c''' = (0,0,0)$.
$c'\times c'' = (0, 0, 2)$, $|c'\times c''| = 2$. $|c'|^2 = 1+4t^2$. $$\kappa = \boxed{2/(1+4t^2)^{3/2}}$$ (与 IV.1 的平面公式一致.) 又 $c''' = 0$ 给 $\det = 0$, 因此 $\boxed{\tau \equiv 0}$.
几何: $\tau\equiv 0$ ⟺ 曲线在固定平面内 (Klingenberg Prop. 1.5.4). 这里曲线在 $xy$-平面, 不出意外. 另外注意: 即使 $c'''$ 不是 $0$, 只要曲线在固定平面里, 三个向量 $c', c'', c'''$ 也共面 ($\det = 0$). 这里 $c''' = 0$ 是凑巧, 不是本质.

§6.4 V.4 — 扭三次曲线 $c(t) = (t, t^2, t^3)$ ("twisted cubic")

展开计算
$c' = (1, 2t, 3t^2), |c'|^2 = 1+4t^2+9t^4$.
$c'' = (0, 2, 6t)$, $c''' = (0, 0, 6)$.
$c'\times c'' = \det\begin{pmatrix}\vec i & \vec j & \vec k\\ 1 & 2t & 3t^2\\ 0 & 2 & 6t\end{pmatrix} = (12t^2 - 6t^2, -(6t - 0), 2 - 0) = (6t^2, -6t, 2)$.
$|c'\times c''|^2 = 36t^4 + 36t^2 + 4 = 4(9t^4 + 9t^2 + 1)$, $|c'\times c''| = 2\sqrt{9t^4+9t^2+1}$. $$\boxed{\,\kappa = \frac{2\sqrt{9t^4+9t^2+1}}{(1+4t^2+9t^4)^{3/2}}\,}$$ $\det(c', c'', c''')$: 注意矩阵 $[c'|c''|c''']$ 是上三角! $$\det\begin{pmatrix}1 & 2t & 3t^2\\ 0 & 2 & 6t\\ 0 & 0 & 6\end{pmatrix} = 1\cdot 2\cdot 6 = 12.$$ 因此 $$\boxed{\,\tau = \frac{12}{4(9t^4+9t^2+1)} = \frac{3}{9t^4+9t^2+1}\,}$$ $t=0$: $\kappa = 2$, $\tau = 3$. $|t|\to\infty$: $\kappa, \tau\to 0$.
反思 (上三角技巧). 看到 $c(t) = (\text{poly}_1(t), \text{poly}_2(t), \text{poly}_3(t))$ 单调升次数的形式, 矩阵 $[c'|c''|c''']$ 总是上三角 (因为 $c^{(k)}$ 的前 $k-1$ 个分量必然消掉 $t$ 项). 这能让 $\det$ 一秒算出. Twisted cubic 是所有非退化空间曲线的局部模型: 任何 $\kappa\ne 0, \tau\ne 0$ 的空间曲线在某点附近的 Taylor 展开看起来都像 twisted cubic.

§7 Part VII — 曲面四例 (4 题)

Part VI 是框架 (只有公式没有题), Part VII 是应用框架到 4 个具体曲面上. 框架回顾:

曲面计算 5 步:
  1. $X_u, X_v$ (一阶偏导, 切向量), 算 $E = X_u\cdot X_u, F = X_u\cdot X_v, G = X_v\cdot X_v$.
  2. $X_u\times X_v$ (法向), 归一化得 $\nu$. 注意符号选择: 是向"外"还是向"里"?
  3. $X_{uu}, X_{uv}, X_{vv}$ (二阶偏导), 与 $\nu$ 点乘得 $L, M, N$.
  4. 若 $F = M = 0$ (正交参数化, 对角情形, 常见): $k_1 = L/E$, $k_2 = N/G$.
  5. 否则用 $K = (LN-M^2)/(EG-F^2)$, $H = (EN+GL-2FM)/(2(EG-F^2))$, $k_{1,2} = H\pm\sqrt{H^2-K}$.

策略提示:

  • 挑参数化很重要: 题目给的是常规参数化, 一般 $F=M=0$ 会化简. 如果不是, 像椭圆抛物面 ($X(x,y) = (x,y, (x^2+y^2)/2)$) 这种, 就只能上 general formula.
  • $\nu$ 的方向影响 $L, M, N$ 的符号, 进而影响 $H$ 的符号 (但不影响 $K$, 因为 $K = k_1 k_2$ 任何两个负数相乘还是正). 一般约定: cross product 给出的方向是"题目认可的".
  • 球面是umbilic ($k_1 = k_2$) 的例子, 平面与圆柱$K = 0$ 但不是平面的例子 (developable surface), 椭圆抛物面是$K > 0$ everywhere 的例子.

§7.1 VII.1 — 平面 $X(u,v) = (u, v, 0)$

展开计算
$X_u = (1,0,0), X_v = (0,1,0)$. $E = 1, F = 0, G = 1$. $X_u\times X_v = (0,0,1)$, $\nu = (0,0,1)$.
所有二阶导都是 $0$: $L = M = N = 0$. 解 $k^2 = 0$: $$\boxed{\,k_1 = k_2 = 0, \quad H = 0, \quad K = 0\,}$$
平面是"完全平的"曲面 — 每个方向曲率都是 0. 反过来: 嵌入 $\R^3$ 的曲面如果 $H \equiv 0$ 且 $K \equiv 0$, 局部必是平面.

§7.2 VII.2 — 圆柱 $X(\theta, z) = (R\cos\theta, R\sin\theta, z)$

展开计算
$X_\theta = (-R\sin\theta, R\cos\theta, 0), X_z = (0,0,1)$. $E = R^2, F = 0, G = 1$.
$X_\theta\times X_z = (R\cos\theta, R\sin\theta, 0)$, $\nu = (\cos\theta, \sin\theta, 0)$ (向外).
$X_{\theta\theta} = (-R\cos\theta, -R\sin\theta, 0), X_{\theta z} = 0, X_{zz} = 0$.
$L = -R\cos^2\theta - R\sin^2\theta = -R$, $M = 0, N = 0$.
$F = M = 0$ 对角情形: $k_1 = L/E = -1/R$, $k_2 = N/G = 0$. $$\boxed{\,k_1 = -1/R,\, k_2 = 0,\, H = -1/(2R),\, K = 0\,}$$
几何 (关键):
  • $K = 0$ 说明圆柱本质上平 ("developable") — 它可以剪开沿一条直线展平成平面, 而不改变长度. Gauss 的 Theorema Egregium 把这个抽象成 "intrinsic" 性质.
  • $k_1 = -1/R$ 在$\theta$ 方向 (绕一圈是圆, 曲率 $1/R$, 符号因 $\nu$ 选择是负), $k_2 = 0$ 在$z$ 方向 (是直线, 不弯).
  • $H = -1/(2R) \ne 0$ 把圆柱与平面区分开 — 平面 $H = K = 0$, 圆柱 $K = 0$ 但 $H \ne 0$.

§7.3 VII.3 — 球面 $X(\theta, \phi) = R(\sin\phi\cos\theta, \sin\phi\sin\theta, \cos\phi)$

展开计算
假设 $0 < \phi < \pi$ 以避开极点.
$X_\theta = R\sin\phi\cdot(-\sin\theta, \cos\theta, 0)$, $X_\phi = R(\cos\phi\cos\theta, \cos\phi\sin\theta, -\sin\phi)$.
$E = R^2\sin^2\phi$, $G = R^2$, $F = 0$ (交叉项相消).
$X_\theta\times X_\phi = -R^2\sin\phi\cdot(\sin\phi\cos\theta, \sin\phi\sin\theta, \cos\phi)$, 模长 $R^2\sin\phi$, $\nu = -(\sin\phi\cos\theta, \sin\phi\sin\theta, \cos\phi) = -X/R$ (向内, 朝原点).
二阶导:
  • $X_{\theta\theta} = (-R\sin\phi\cos\theta, -R\sin\phi\sin\theta, 0)$; $L = X_{\theta\theta}\cdot\nu = R\sin^2\phi(\cos^2\theta + \sin^2\theta) = R\sin^2\phi$.
  • $X_{\theta\phi} = R\cos\phi\cdot(-\sin\theta, \cos\theta, 0)$, 计算 $M = 0$.
  • $X_{\phi\phi} = -X$, 所以 $N = -X\cdot\nu = -X\cdot(-X/R) = |X|^2/R = R$.
$F = M = 0$: $k_1 = L/E = R\sin^2\phi/(R^2\sin^2\phi) = 1/R$. $k_2 = N/G = R/R^2 = 1/R$. $$\boxed{\,k_1 = k_2 = 1/R,\, H = 1/R,\, K = 1/R^2\,}$$
几何 (重要):
  • $k_1 = k_2$ 说明球面是umbilic (每点每个方向曲率都一样). 这是 umbilic surface 的极致.
  • $K = 1/R^2 > 0$ 常数 — 球面是常正高斯曲率曲面 (常曲率).
  • $\nu$ 选向内还是向外, $H$ 的符号会翻 (但 $K$ 不变, 因为 $K = k_1 k_2$, 两个 sign flip 后乘积不变). 这是 Theorema Egregium 的简单实例.
  • 分类定理: 连通完备 umbilic 曲面 (in $\R^3$) 必是球面或平面. 没有"中间"形态.

§7.4 VII.4 — 椭圆抛物面 $X(x,y) = (x, y, (x^2+y^2)/2)$

策略 (与前三题不同). 这是 4 题里唯一 $F \ne 0$ 的, 不能用对角公式. 必须用 general formula $K = (LN-M^2)/(EG-F^2)$, $H = (EN+GL-2FM)/(2(EG-F^2))$.

展开计算 (含主曲率简化)
$X_x = (1, 0, x), X_y = (0, 1, y)$.
$E = 1 + x^2, F = xy, G = 1 + y^2$. 令 $w := 1 + x^2 + y^2$. 然后 $EG - F^2 = (1+x^2)(1+y^2) - x^2y^2 = 1 + x^2 + y^2 = w$.
$X_x\times X_y = (-x, -y, 1)$, $|X_x\times X_y| = \sqrt{w}$, $\nu = (-x, -y, 1)/\sqrt{w}$.
$X_{xx} = (0,0,1), X_{xy} = 0, X_{yy} = (0,0,1)$.
$L = 1/\sqrt w, M = 0, N = 1/\sqrt w$.
Gauss 曲率: $$K = \frac{LN - M^2}{EG - F^2} = \frac{1/w}{w} = \boxed{\,\frac{1}{(1+x^2+y^2)^2}\,}$$ 平均曲率: $$H = \frac{EN + GL - 2FM}{2(EG-F^2)} = \frac{(1+x^2)\cdot\frac{1}{\sqrt w} + (1+y^2)\cdot\frac{1}{\sqrt w}}{2w} = \boxed{\,\frac{2+x^2+y^2}{2(1+x^2+y^2)^{3/2}}\,}$$ 主曲率简化: 设 $r^2 = x^2+y^2$, $w = 1+r^2$. 算 $H^2 - K = \frac{(2+r^2)^2}{4w^3} - \frac{1}{w^2} = \frac{(2+r^2)^2 - 4w}{4w^3} = \frac{r^4}{4w^3}$. 因此 $\sqrt{H^2-K} = r^2/(2w^{3/2})$, 给出 $$\boxed{\,k_1 = \frac{1+r^2}{w^{3/2}} = \frac{1}{\sqrt w} = \frac{1}{\sqrt{1+x^2+y^2}},\quad k_2 = \frac{1}{w^{3/2}} = \frac{1}{(1+x^2+y^2)^{3/2}}\,}$$ 验证: $k_1 k_2 = 1/w^2 = K$ ✓.
几何:
  • 原点 $(0,0)$: $E = G = 1, F = 0, L = N = 1, M = 0$ — 第一与第二基本形式相等, 故 $k_1 = k_2 = 1$. 椭圆抛物面在顶点与单位球相切并曲率相等.
  • $K > 0$ everywhere — 椭圆抛物面是处处局部凸的曲面.
  • 这是 4 题里唯一 $F\ne 0$ 的, 训练你什么时候不能用对角公式. 如果你看到 $F = xy \ne 0$, 立刻拿出 general formula.

§8 Part VIII — 环面 (Torus) Gauss 曲率符号

Problem VIII. 考虑环面 $$X(u, v) = ((R + r\cos v)\cos u, (R + r\cos v)\sin u, r\sin v), \quad R > r > 0.$$ 计算主曲率、平均曲率、Gauss 曲率. 判定 Gauss 曲率正/负/零的区域.

策略 (重头戏). Part VIII 的 1 道题 = Part VII 4 道题的"压轴版", 而且加了一个定性分析 ($K$ 的符号在哪里正/负/零, 这考你几何理解而不只是计算).

  • 设 $\rho := R + r\cos v$ — 这是环面上点到 $z$-轴的距离. 它会反复出现, 提前命名.
  • 套 5 步框架: $X_u, X_v$ → $E, F, G$ → $X_u\times X_v$ → $\nu$ → 二阶 → $L, M, N$.
  • 由参数化的正交性 ($u, v$ 几何上正交), 必有 $F = 0$ 与 $M = 0$. 对角公式!
  • $K = k_1 k_2 = (-\cos v/\rho)\cdot(-1/r) = \cos v/(r\rho)$. 分母总正, 所以 $\operatorname{sgn}(K) = \operatorname{sgn}(\cos v)$.
  • $K$ 的符号分析: $K > 0$ ⟺ $v \in (-\pi/2, \pi/2)$ (外侧, 椭圆点), $K = 0$ ⟺ $v = \pm\pi/2$ (顶/底圆周, parabolic), $K < 0$ ⟺ $v \in (\pi/2, 3\pi/2)$ (内侧, 鞍点).
展开完整计算 (5 步)
Step 1 ($X_u, X_v$, 第一基本形式). 设 $\rho := R + r\cos v$. $$X_u = (-\rho\sin u, \rho\cos u, 0), \quad X_v = (-r\sin v\cos u, -r\sin v\sin u, r\cos v).$$ $$E = \rho^2, \quad G = r^2, \quad F = \rho r\sin u\sin v\cos u - \rho r\sin v\sin u\cos u + 0 = 0.$$ Step 2 ($\nu$). $$X_u\times X_v = \rho r(\cos u\cos v, \sin u\cos v, \sin v), \quad |X_u\times X_v| = \rho r, \quad \nu = (\cos u\cos v, \sin u\cos v, \sin v).$$ (向外法向 — 指向远离 tube 中心线.) Step 3 (二阶偏导). $$X_{uu} = (-\rho\cos u, -\rho\sin u, 0),$$ $$X_{uv} = (r\sin v\sin u, -r\sin v\cos u, 0),$$ $$X_{vv} = (-r\cos v\cos u, -r\cos v\sin u, -r\sin v).$$ Step 4 ($L, M, N$). $$L = X_{uu}\cdot\nu = -\rho\cos u\cdot\cos u\cos v - \rho\sin u\cdot\sin u\cos v = -\rho\cos v.$$ $$M = X_{uv}\cdot\nu = r\sin v\cos v\sin u\cos u - r\sin v\cos v\cos u\sin u = 0.$$ $$N = X_{vv}\cdot\nu = -r\cos^2 v - r\sin^2 v = -r.$$ Step 5 (主/平均/Gauss). $F = M = 0$ 对角: $$\boxed{k_1 = L/E = -\cos v/\rho = -\frac{\cos v}{R+r\cos v}}, \quad \boxed{k_2 = N/G = -1/r}.$$ $$\boxed{\,H = -\frac{R + 2r\cos v}{2r(R + r\cos v)}, \quad K = \frac{\cos v}{r(R+r\cos v)}\,}$$ Sign 分析. 分母 $r(R+r\cos v) > 0$ 总. 所以 $\sgn K = \sgn(\cos v)$:
  • $K > 0$ ($v \in (-\pi/2, \pi/2)$): 外侧 (椭圆点) — 离 $z$ 轴最远的半圈, 双方向都"鼓出去", sphere-like, 正曲率.
  • $K = 0$ ($v = \pm\pi/2$): 顶/底 parabolic circles — 最高与最低纬度圈, $k_1 = 0$ 沿 $u$ 方向是水平直线 (准确地: 切于"上下圆"的切线).
  • $K < 0$ ($v \in (\pi/2, 3\pi/2)$): 内侧 (双曲点 / 鞍点) — 离 $z$ 轴最近的半圈, sphere-bulge 与 saddle 的混合, 两个主方向曲率符号相反.
反思 (Gauss-Bonnet 的体现). 在整个环面上积分 Gauss 曲率: $$\iint_T K\,dS = \int_0^{2\pi}\int_0^{2\pi} \frac{\cos v}{r(R+r\cos v)}\cdot \sqrt{EG-F^2}\,du\,dv.$$ $\sqrt{EG-F^2} = \rho r = r(R + r\cos v)$ 正好把 $K$ 的分母约掉, 留下 $\int\int \cos v\,du\,dv = 0$. 这正是 Gauss-Bonnet 定理 的实例: $\iint_M K\,dS = 2\pi\chi(M)$. 环面的 Euler characteristic $\chi(T) = 0$, 所以 $\iint K\,dS = 0$. 正曲率区域 (外侧) 与负曲率区域 (内侧) 在积分意义下"等量抵消".
题外话 (定性图像):
  • 把环面想象成甜甜圈. 外侧表面 = 你舔甜甜圈外圈的那一面 (像球). 内侧表面 = 甜甜圈的"洞"那一面 (像鞍).
  • $v = 0$ 对应正外侧最远点 (椭圆); $v = \pi$ 对应正内侧最近点 (双曲); $v = \pm\pi/2$ 对应正上 / 正下 (parabolic).
  • $k_2 = -1/r$ 在所有点都是常数 — 因为 $v$ 方向是绕 tube 一圈的小圆, 曲率永远是 $1/r$ (符号因 $\nu$ 选择).
  • $k_1 = -\cos v/\rho$ 变号, 反映了 longitudinal direction (绕 $z$ 轴) 在外侧"向 $\nu$ 反方向凸", 在内侧"向 $\nu$ 同方向凸".

§9 方法论速查

🟦 Fourier 标准变换 (背)

  • $\widehat{\1_{[-a,a]}}(\xi) = \sin(2\pi a\xi)/(\pi\xi)$ — sinc, $1/|\xi|$ decay
  • $\widehat{e^{-a|x|}}(\xi) = 2a/(a^2 + 4\pi^2\xi^2)$ — Lorentzian, $1/\xi^2$ decay
  • $\widehat{e^{-\pi x^2}}(\xi) = e^{-\pi\xi^2}$ — self-dual
  • Heat kernel $G_t$: $\widehat{G_t}(\xi) = e^{-4\pi^2 t\xi^2}$
  • 张量积律: $\widehat{f\otimes g} = \widehat{f}\otimes\widehat{g}$ (Fubini)

🟦 Laplace 标准变换 (背)

  • $\LT(1) = 1/s$, $\LT(t^n) = n!/s^{n+1}$, $\LT(e^{at}) = 1/(s-a)$
  • $\LT(\sin bt) = b/(s^2+b^2)$, $\LT(\cos bt) = s/(s^2+b^2)$
  • Shift: $\LT(e^{at}g(t))(s) = \LT(g)(s-a)$
  • 导数律: $\LT(y') = sY - y(0)$, $\LT(y'') = s^2 Y - sy(0) - y'(0)$

🟨 变换求解 PDE/ODE 范式

  • PDE on $\R\times[0,\infty)$ via FT: 对空间 FT → 留下 $t$-ODE → 解 → 反 FT 识别 convolution
  • 纯代数 PDE on $\R$ via FT: 变代数除法 → 识别 kernel → convolution with $f$
  • ODE on $[0,\infty)$ via LT: $\LT$ → 部分分式 → 查表反演
  • 共振 ($1/(s^2+b^2)^2$ 出现): 用卷积定理反演; 解里有 $t\sin(bt)$ 或 $t\cos(bt)$ 项
  • $(LN-M^2)$ 题型: 公式 + 简化, 看是否对角

🟢 曲率公式 (背)

  • 平面曲线: $\kappa = |x'y'' - y'x''|/((x')^2+(y')^2)^{3/2}$
  • 空间曲线: $\kappa = |c'\times c''|/|c'|^3$; $\tau = \det(c',c'',c''')/|c'\times c''|^2$
  • 螺旋 $(R\cos t, R\sin t, ht)$: $\kappa = R/(R^2+h^2)$, $\tau = h/(R^2+h^2)$ (常数)
  • 曲面 (orthogonal, $F=M=0$): $k_1 = L/E$, $k_2 = N/G$, $H = (k_1+k_2)/2$, $K = LN/EG$
  • 曲面 (general): $K = (LN-M^2)/(EG-F^2)$, $H = (EN+GL-2FM)/(2(EG-F^2))$, $k_{1,2} = H\pm\sqrt{H^2-K}$

🟢 4 个标准曲面的曲率 (记)

  • 平面: $E=G=1, F=L=M=N=0$, $H = K = 0$
  • 圆柱 $R$: $E=R^2, G=1$, $L=-R, N=0$, $k_1=-1/R, k_2=0$, $K = 0$ (developable)
  • 球 $R$: $E=R^2\sin^2\phi, G=R^2$, $L=R\sin^2\phi, N=R$, $k_1=k_2=1/R$ (umbilic), $K = 1/R^2$
  • 椭圆抛物面 $z = (x^2+y^2)/2$: $K = 1/(1+r^2)^2$, 顶点 $K = 1$
  • 环面 $(R, r)$: $k_1 = -\cos v/\rho, k_2 = -1/r$, $K = \cos v/(r\rho)$ 符号随 $v$ 变

🟥 易错点

  • "FT 微分律的符号" — 用 Hubbard 约定 $e^{+2\pi i x\xi}$ 时 $\widehat{f'} = -2\pi i\xi\widehat{f}$. 用错 sign 公式整题翻车.
  • "Lebesgue/Riemann 同名不同事" — Practice Final 里 LT/FT 都是普通 Riemann 积分, 不是 MCT/DCT 主战场. 但积分号与导数号交换用的是 DCT (DUI), 这里要严谨.
  • "曲面参数化的 $F = ?$" — 检查 $F = X_u\cdot X_v$. 椭圆抛物面 $F = xy \ne 0$ 不能用对角公式!
  • "$\nu$ 的方向" — $X_u\times X_v$ 可能向"外"也可能向"里". 这影响 $L, M, N$ 与 $H$ 的符号. 但不影响 $K$.
  • "忘记验证" — ODE 解出来代回原方程 + 初值条件检验! 部分分式系数错一个就全错.
  • "二阶 ODE 共振时" — 看到 $\sin t / \cos t$ 强迫加 $y''+y$ 这种, 知道解里必有 secular term $t\cos t$ 或 $t\sin t$.
