§2 Problem 1 — $\Q \cap [0,1]$: measure zero + undefined volume

📋 Problem 1

找一个 measure zero 的 $\R^n$ 子集 $X$, 它有undefined volume. 证明 $X$ 既 measure zero 又 undefined volume.

策略

这是 §3 (measure-zero vs vol-zero) 的标志性反例问题. 直觉 + 设计:

  1. 需要的几何性质: $X$ 既要 measure 0 (i.e. 能被任意小的countable cube cover 覆盖) 又要 vol undefined (i.e. $\mathbf 1_X$ 不可积).
  2. "countable" 是 measure-0 的工具, "dense" 是 vol-undef 的工具: 一个 countable 集合自然 measure 0 (P3 的结论, 用 $\eps/2^j$-trick); 一个 dense 集合让 $\mathbf 1_X$ 的 $U_N - L_N \not\to 0$ (类比 HW1 P1a 的 Dirichlet 反例).
  3. 所以需要 countable AND dense in 某区间 的集合. 最自然的候选: $\Q \cap [0, 1]$.

这是 measure-0 与 vol-undef 的 "regulation example" — 同时具备两个 ostensibly 相反的属性. 它出现的根本原因是: countable (measure-0 的 ε/2^j 工具适用) 与 dense (Riemann sum 视角看不出 X 与 complement 的区别) 在 $\Q \cap [0,1]$ 上共存 — 是 $\Q$ 在 $\R$ 中的代数-拓扑 双重身份.

展开严格证明 (双向: measure 0 + vol undefined)
证明

取 $X := \Q \cap [0, 1] \subset \R$ (一般 $\R^n$ 情形可取 $X \times \{0\}^{n-1}$ 或 $\Q^n \cap [0,1]^n$).

(1) $X$ measure zero: $X$ countable (subset of countable $\Q$). 枚举 $X = \{q_1, q_2, q_3, \ldots\}$. 对 $\eps > 0$, 取开 cube $$B_j := (q_j - \eps/2^{j+2},\;\; q_j + \eps/2^{j+2}), \qquad \vol_1(B_j) = \eps/2^{j+1}.$$ $X \subseteq \bigcup_j B_j$ (每个 $q_j \in B_j$), 总体积 $$\sum_j \vol_1(B_j) = \sum_j \eps/2^{j+1} = \eps/2 < \eps.$$ By Def 3.1.1, $X$ has measure zero. $\checkmark$

(2) $X$ has undefined volume: 等价于 $\mathbf 1_X \notin \IR$. 用 Cauchy 准则: 显示 $U_N(\mathbf 1_X) - L_N(\mathbf 1_X) \ge 1$ for all $N$.

固定 $N$, 考虑 dyadic paving $\mathcal D_N(\R)$. 令 $\mathcal C_N := \{C \in \mathcal D_N : C \cap [0, 1] \ne \emptyset\}$.

对每个 $C \in \mathcal C_N$ (dyadic 区间 of length $1/2^N$, 与 $[0,1]$ 相交): $C \cap [0, 1]$ nonempty + nondegenerate, 由 $\Q$ 稠密 ⟹ 含 rational ⟹ $\sup_C \mathbf 1_X = 1$. 对 $C \notin \mathcal C_N$ (即 $C$ 与 $[0,1]$ 不相交), $\mathbf 1_X \equiv 0$ on $C$, 故 $\sup_C \mathbf 1_X = 0$.

$$U_N(\mathbf 1_X) = \sum_{C \in \mathcal C_N}\vol_1(C) \ge 1$$ (cubes in $\mathcal C_N$ 覆盖 $[0, 1]$, 总长 $\ge 1$).

对每个 $C \in \mathcal D_N$ (任意 dyadic 区间, positive length): 由 $\R \setminus \Q$ 稠密 ⟹ $C$ 含 irrational ⟹ $\inf_C \mathbf 1_X = 0$. $$L_N(\mathbf 1_X) = 0.$$

故对所有 $N$, $U_N(\mathbf 1_X) - L_N(\mathbf 1_X) \ge 1$, 违反 Cauchy 准则. $\mathbf 1_X \notin \IR$, $\vol_1(X)$ undefined. $\blacksquare$

💡 反思 — measure zero 与 vol zero 的"双向 gap"

两个方向都失败: P1 给出"measure 0 但 vol 未定义"的例子. 反方向呢? "vol 0 但不 measure 0"? 不可能: 若 $\vol(X) = 0$, 则 $\mathbf 1_X \in \IR$, 进一步 (由 HW1 P4a Prop 4.1.23) "$X$ 能被有限 dyadic cube 覆盖, 总体积任意小". 有限 cover trivial 也是 countable cover, 故 measure 0. 故 vol 0 ⟹ measure 0 always.

蕴含关系: vol 0 ⊊ measure 0 ⊊ {bounded sets}. P1 的 $\Q \cap [0, 1]$ 落在 "measure 0 \ vol 0" 的 strict gap 里. 这个 gap 是 Lebesgue 积分发明的历史动机 — 它把 $\Q \cap [0,1]$ "合并" 到可积函数定义里 (Lebesgue 积分下 $\int \mathbf 1_{\Q \cap [0,1]} dV_L = 0$). 见 Topic 12.

$U_N - L_N \ge 1$ 是极限失败的 sharp 量化: 不是"$U_N - L_N$ 慢慢趋 0"或"趋某 small 正数", 而是下界恒 $\ge 1$. 这反映 $\mathbf 1_X$ 在每个 dyadic level 都是 wildly oscillating, 加 partition 没用 — 与连续函数 (uniform continuity 让 oscillation $\to 0$) 形成最尖锐对比.

§3 Problem 2 — 连续 + bounded support ⟹ 可积 (用 upper-lower sum)

📋 Problem 2

$f \colon \R^n \to \R$ bounded continuous with bounded support. 证明 $f$ 可积, 用 upper-lower sum.

策略

这个 statement 本身可由 Lebesgue 准则立即推出 ($\Disc f = \emptyset$ measure zero) — 这是 HW1 P1b 走的路. 但本题规定不用 Lebesgue 准则, 要 "用 upper-lower sum" 直接证明. 这是为了训练你 raw 操作 dyadic paving 的能力, 不依赖高阶定理.

关键挑战: $f$ 的 support 是bounded, 但不一定是 dyadic cube — 它可能是 ball 或 irregular 集. 我们怎么把 "$f$ 在 $\supp(f)$ 上 uniform continuous" 翻译成 "$U_N(f) - L_N(f)$ 小"?

核心 trick (case-by-case): 对每个 dyadic cube $C \in \mathcal D_N$:

  • (a) $C \subseteq K$ (包含 supp 的 fattened compact box): 用 uniform continuity ⟹ $\osc_C(f) \le \eps$.
  • (b) $C \not\subseteq K$: $C$ 触出 $K$ 外, 结合 small diameter + $K$ 是 $\supp(f)$ 的 fattening ⟹ 整个 $C$ 落在 $\supp(f)$ 外, $f \equiv 0$ on $C$, $\osc_C(f) = 0$.

取 $K = [-R-1, R+1]^n$ (其中 $R$ 是 $\supp(f) \subseteq [-R, R]^n$). 取 $N$ 使 $\diam(C_{\mathbf k, N}) = \sqrt n/2^N < \min(\delta, 1)$ (其中 $\delta$ 来自 uniform continuity). "$< \delta$" 处理 case (a) 的 oscillation; "$< 1$" 处理 case (b) 的 "$C$ 整个在 $K$ 外". 两个条件叠加, $U_N(f) - L_N(f) \le \eps \cdot (2R+2)^n$.

展开严格证明 (Heine-Cantor + dyadic + case-by-case)
证明

Setup: $\supp(f) \subseteq [-R, R]^n$ for some $R > 0$. 设 $K := [-R-1, R+1]^n$ (compact). $f$ continuous on $K$ ⟹ uniformly continuous on $K$ (Heine-Cantor).

对 $\eps > 0$, $\exists \delta > 0$ 使 $$x, y \in K,\;\norm{x - y} < \delta \;\Longrightarrow\; \abs{f(x) - f(y)} < \eps. \qquad (\star)$$ 取 $N$ 满足 $\diam(C_{\mathbf k, N}) = \sqrt n / 2^N < \min(\delta, 1)$.

Claim: 对每个 $C \in \mathcal D_N$, 恰好一个 case 成立:

  1. $C \subseteq K$ ⟹ $\osc_C(f) \le \eps$.
  2. $C \not\subseteq K$ ⟹ $f \equiv 0$ on $C$, 故 $\osc_C(f) = 0$.

证 (a): $C \subseteq K$ ⟹ 任意 $x, y \in C$ 满足 $x, y \in K$ 且 $\norm{x - y} \le \diam C < \delta$. By $(\star)$, $\abs{f(x) - f(y)} < \eps$. Sup over $x, y \in C$: $\osc_C(f) \le \eps$. $\checkmark$

证 (b): $C \not\subseteq K$ ⟹ $\exists y \in C \setminus K$, i.e., 某 coord $\abs{y_i} > R + 1$. 设 $y_i > R + 1$. 对任意 $z \in C$, $\abs{z_i - y_i} \le \diam C < 1$, 故 $z_i > y_i - 1 > R$, 即 $z \notin [-R, R]^n \supseteq \supp(f)$. 故 $f(z) = 0$. $z \in C$ 任意, $f \equiv 0$ on $C$. $\checkmark$

组合: $C \subseteq K$ 的 cubes 总体积 $\le \vol(K) = (2R + 2)^n$. 故 $$U_N(f) - L_N(f) = \sum_{C \in \mathcal D_N}\osc_C(f)\cdot\vol(C) = \sum_{C \subseteq K}\osc_C(f)\cdot\vol(C) + \underbrace{\sum_{C \not\subseteq K}\osc_C(f)\cdot\vol(C)}_{= 0}$$ $$\le \eps \cdot \sum_{C \subseteq K}\vol(C) \le \eps \cdot (2R + 2)^n.$$ 对任意 $\eps' > 0$, 取 $\eps = \eps'/(2R+2)^n$ ⟹ $U_N(f) - L_N(f) < \eps'$. By Cauchy 准则, $f \in \IR$. $\blacksquare$

💡 反思 — "$< \delta$" + "$< 1$" 两个条件的独立角色

diameter 选择 $\sqrt n/2^N < \min(\delta, 1)$ 是同时满足两个条件的要求:

  • $< \delta$: 用于 case (a). uniform continuity 的 $\delta$ 让 $\osc_C(f) \le \eps$ 在 $C \subseteq K$ 时成立.
  • $< 1$: 用于 case (b). $K = [-R-1, R+1]^n$ 比 $\supp(f) \subseteq [-R, R]^n$ "胖" $1$ 个单位; cube diameter $< 1$ 让 $C \not\subseteq K$ 时 $C$ 整体落在 $\supp f$ 外.

如果只用 $< \delta$ 而不要求 $< 1$, case (b) 可能 contain $C$ 一部分 in $\supp f$ 一部分外, $f$ 在 $C$ 上 oscillation 不一定 0 也不一定 $\le \eps$ — 失控. 故 $1$ 这个数字不是"魔法常数", 而是$K$ 与 $\supp f$ 之间的 安全 margin.

对比 Lebesgue 准则 short-cut: 用 Lebesgue 准则只要"$\Disc f = \emptyset$ measure 0", 一句话 done. 但 Lebesgue 准则本身的证明 (Lec 3.2) 需要类似 P2 的技术 + 测度零的 cover 论证. 所以 "用 upper-lower sum 直接" 实际是不调用 Lebesgue 准则的 self-contained 证明, 更"raw" 但同样严格. 这也是为什么 P2 在 HW2 里出现 — 它训练 mid-level 技术不依赖 high-level 定理.

P3 — Countable subset of $\R^n$ has measure zero

📋 Problem 3

证明任意 countable subset of $\R^n$ has measure zero.

策略 + 证明

核心是 $\eps/2^j$-trick: 对 countable family $\{x^{(j)}\}_{j=1}^\infty$, 给每个 $x^{(j)}$ 配一个体积 $\eps/2^{j+1}$ 的 cube. 求和: $\sum_j \eps/2^{j+1} = \eps/2 < \eps$. 这一招在 §3-§4 反复出现 (countable union of measure-zero sets, P1, P4 等), 是 measure 理论的瑞士军刀.

展开证明
证明

$S \subseteq \R^n$ countable, 枚举 $S = \{x^{(1)}, x^{(2)}, \ldots\}$ (有限或可数). 对 $\eps > 0$, 设 $s_j := (\eps/2^{j+1})^{1/n}$, 取以 $x^{(j)}$ 为中心、边长 $s_j$ 的开 cube $$B_j := \prod_{i=1}^n (x^{(j)}_i - s_j/2,\;\; x^{(j)}_i + s_j/2), \qquad \vol_n(B_j) = s_j^n = \eps/2^{j+1}.$$ $\{B_j\}$ countable cover of $S$ (每个 $x^{(j)} \in B_j$), 总体积 $$\sum_j \vol_n(B_j) = \sum_j \eps/2^{j+1} = \eps/2 < \eps.$$ By Def 3.1.1, $S$ has measure zero. $\blacksquare$

立得推论: $\Q^n \subseteq \R^n$ has measure zero (因为 $\Q^n$ countable). 也包含 algebraic numbers, finite 集, sequence 极限的有限集合等.

P4 — Thomae's function: integrable, discontinuous on undefined-vol set

📋 Problem 4

找 integrable $f \colon \R \to \R$ which is discontinuous on a set of undefined volume.

策略

我们需要的特征: (1) $f \in \IR$, (2) $\Disc f$ has undefined volume (即 $\mathbf 1_{\Disc f} \notin \IR$). 由 (1) Lebesgue 准则 ⟹ $\Disc f$ measure 0; 由 (2) 又 $\Disc f$ 不能太规则 (否则 $\mathbf 1_{\Disc f}$ 会可积). 所以 $\Disc f$ 必须是 measure 0 但 vol-undef 的集合 — 由 P1, 这就是 $\Q \cap [-1, 1]$ 类型的集合.

但有这种 $\Disc$ 的 integrable 函数不容易构造. 一般思路: 不能用 $\mathbf 1_{\Q \cap [-1, 1]}$ 因为这个 indicator 本身不可积. 需要找一个 "$\Disc = \Q \cap [-1, 1]$ 但 $f$ 是可积的" 函数. 这就是经典的 Thomae's function (popcorn function, ruler function): $$f(x) = \begin{cases} 1/q & x = p/q \in \Q \cap [-1, 1] \text{ in lowest terms} \\ 0 & \text{otherwise}\end{cases}$$

关键性质 (Lec 3.2 Example 3.2.1 完整证):

  • $f$ continuous at every irrational (and outside $[-1, 1]$): 由有限性论证 — $\{q : 1/q \ge \eps\}$ 是 finite set, 故 $\{p/q \in [-1, 1] : 1/q \ge \eps\}$ 也 finite, irrational point 与这个 finite set 的距离 $\ge \delta > 0$.
  • $f$ discontinuous at every rational: $f(p/q) = 1/q > 0$, but irrationals dense ⟹ exists irrational sequence $\xi_k \to p/q$ with $f(\xi_k) = 0$. 故 $\Disc f = \Q \cap [-1, 1]$.
  • $\Disc f$ measure zero (by P3, countable ⟹ measure 0) ⟹ $f$ integrable (Lebesgue 准则).
  • $\vol(\Disc f) = \vol(\Q \cap [-1, 1])$ undefined (by P1, same density argument).

这就完美命中题目要求.

展开严格证明 (Thomae continuity analysis + P3 + P1 调用)
证明

定义: $f \colon \R \to \R$: $$f(x) := \begin{cases}1/q & x \in \Q \cap [-1, 1],\;x = p/q \text{ in lowest terms},\;q \ge 1, \\ 0 & \text{otherwise}.\end{cases}$$ (约定 $0 = 0/1$, 故 $f(0) = 1$. $\supp(f) \subseteq [-1, 1]$, $0 \le f \le 1$ — bounded with bounded support.)

Claim 1: $\Disc f = \Q \cap [-1, 1]$.

(Discontinuity at rationals): $x_0 = p_0/q_0 \in \Q \cap [-1, 1]$, $f(x_0) = 1/q_0 > 0$. Irrationals dense ⟹ $\exists \xi_k \to x_0$, $\xi_k \in [-1, 1] \setminus \Q$, $f(\xi_k) = 0$. Hence $\lim f(\xi_k) = 0 \ne 1/q_0 = f(x_0)$. Discontinuous.

(Continuity at irrationals + outside $[-1, 1]$): 设 $x_0 \in [-1, 1] \setminus \Q$ (irrational case; outside $[-1, 1]$ trivially 由 local zero). $f(x_0) = 0$. 对 $\eps > 0$:

  • 有限性 trick: $Q_\eps := \{q \in \N : 1/q \ge \eps\} = \{1, 2, \ldots, \lfloor 1/\eps \rfloor\}$ 是finite.
  • $S_\eps := \{p/q \in [-1, 1] : q \in Q_\eps,\;\gcd(p, q) = 1\}$ 也 finite (每个 $q \in Q_\eps$ 给 $\le 2q + 1$ 个 valid $p$).
  • $x_0$ irrational ⟹ $x_0 \notin S_\eps$, 故 $\delta := \min_{s \in S_\eps}\abs{x_0 - s} > 0$.

对 $x \in (x_0 - \delta, x_0 + \delta)$:

  • $x$ irrational 或 $\abs x > 1$: $f(x) = 0$, $\abs{f(x) - f(x_0)} = 0 < \eps$.
  • $x = p/q \in \Q \cap [-1, 1]$ in lowest terms: $x \notin S_\eps$ (by $\delta$ choice) ⟹ $q \notin Q_\eps$ ⟹ $1/q < \eps$. 故 $f(x) = 1/q < \eps$.
两种情况都 $\abs{f(x) - f(x_0)} < \eps$. Continuity at $x_0$. $\checkmark$

Claim 2: $f$ integrable. 由 Claim 1, $\Disc f = \Q \cap [-1, 1]$ countable. 由 P3, countable ⟹ measure 0. By Lebesgue 准则 (Thm 4.1.13), $f \in \IR(\R)$. $\checkmark$

Claim 3: $\Disc f = \Q \cap [-1, 1]$ has undefined volume. 由 P1 同款证: 每个 dyadic interval intersecting $[-1, 1]$ 含 rational ⟹ $M_C(\mathbf 1_{\Disc f}) = 1$, 求和 $\ge 2$; 每个 dyadic interval 含 irrational ⟹ $m_C(\mathbf 1_{\Disc f}) = 0$. $U_N - L_N \ge 2$ for all $N$ ⟹ $\mathbf 1_{\Disc f} \notin \IR$ ⟹ vol undefined. $\checkmark$

Combine: $f$ integrable, $\Disc f$ undefined vol. $\blacksquare$

💡 反思 — Thomae 的几何 + P3+P4 联手反映 Lebesgue 准则的"本质"

Thomae 的几何形象: $f$ 在每个 rational $p/q$ 上有一个 spike of height $1/q$. 小 denominator (e.g. $1/2, 1/3, 2/3$) 给大 spike; 大 denominator (e.g. $123/456$) 给 tiny spike. 图象长得像 "popcorn" (爆米花) — 离散的 spike 越靠近 $0$ 越密集 (rationals are dense), 但大 spike (即大 $1/q$) 稀疏 (它们对应 simple fraction). 这就是为什么 $f$ 在 irrationals 处连续: 任意小邻域里, 大 spike 只有有限多, 故 $\delta$ 取得 small enough 就避开了它们.

P3 + P4 联手反映 Lebesgue 准则的核心: Lebesgue 准则告诉我们 "discontinuity 集合measure 0 就够 (不需要 vol 0 或 closed 等更强属性)". P3 + P4 联合给出一个 dramatic 验证 —

  • P3: 任何 countable 集 measure 0.
  • P4: $\Disc(\text{Thomae}) = \Q \cap [-1, 1]$ countable, 故 measure 0, 故 Thomae integrable. 尽管 $\Disc f$ 是 dense + undefined volume (P1 的属性).

"Discontinuity 集合即使 dense, 即使 vol-undefined, 只要 countable 仍 measure 0, 仍允许函数可积" — 这是 §4.1 整章的顶点论述. 把 Riemann 可积性从"$\Disc$ 必须是稀薄几何对象"放宽到"$\Disc$ 必须 measure 0 (拓扑可以任意复杂)". 见 Topic 06.

对应 §4.11 (Lebesgue 积分): 在 Lebesgue 积分框架下, Dirichlet $\mathbf 1_{\Q \cap [0, 1]}$ 也可积 (积分 = 0). Riemann 积分对 Dirichlet 失败的原因不是 measure-theoretic 而是 "Riemann 定义只能 sum-then-limit, 无法 a.e. 等价 functions". §4.11 通过砖块分解扩张 $\IR \subsetneq \IL$, "修复" Dirichlet 问题. P4 (Thomae) 是 §4.1 框架下 closest you can get to "Dirichlet" 但仍可积的函数.

§5 Problem 4.3.4 — Cube 上对称函数 vs Simplex 上 (a + b)

串联

这道题分两 part: (a) 是结构定理 "cube 上对称函数 = $n!$ × ascending simplex 上", (b) 是 (a) 应用于 product function $F(x) = \prod_i f(x_i)$ 得 $\int_P F = \tfrac{1}{n!}(\int_a^b f)^n$ 闭式公式. (a) 是 main work, (b) 是 elegant corollary.

Part (a) — 排列群分解 $\int_Q f = n! \int_P f$

📋 Problem 4.3.4(a)

$Q := \{x \in \R^n : a \le x_i \le b \forall i\}$ (cube); $P := \{x \in Q : a \le x_1 \le \cdots \le x_n \le b\}$ (ascending simplex). 若 $f$ integrable + symmetric (即 $f(x_1, \ldots, x_n) = f(x_{\sigma(1)}, \ldots, x_{\sigma(n)})$ 对所有 $\sigma \in S_n$), 证 $$\int_Q f = n! \int_P f.$$

策略

从几何分解出发: 把 cube $Q$ 用每种排序切分: $$Q = \bigcup_{\sigma \in S_n} P_\sigma, \qquad P_\sigma := \{x \in Q : x_{\sigma(1)} \le \cdots \le x_{\sigma(n)}\}.$$ 每个 $P_\sigma$ 是一个"permuted simplex", 共 $n!$ 个. 它们 almost-disjoint: $P_\sigma \cap P_\tau$ ($\sigma \ne \tau$) 落在对角集 $B := \bigcup_{i < j}\{x_i = x_j\}$ 里 — 这是 finite union of $(n-1)$-dim hyperplanes, measure zero (Prop 3.2.2).

关键 lemma — coordinate-permutation invariance of integral: 对 $\sigma \in S_n$ 的"坐标置换" 线性映射 $T_\sigma \colon \R^n \to \R^n$ ($(T_\sigma x)_{\sigma(i)} = x_i$), $\int g \circ T_\sigma = \int g$ 对任意 integrable $g$. 直观: $T_\sigma$ permutes dyadic cubes, 保 vol, 故 $U_N(g \circ T_\sigma) = U_N(g)$ via reindexing.

组合: $f$ symmetric ⟹ $f \circ T_\sigma = f$ pointwise ⟹ $\int_{P_\sigma} f = \int_{T_\sigma^{-1}(P_\sigma)} f \circ T_\sigma = \int_P f$ (因 $T_\sigma^{-1}(P_\sigma) = P$, 即 ascending simplex). 求和 over $\sigma \in S_n$: $$\int_Q f = \sum_\sigma \int_{P_\sigma} f = \sum_\sigma \int_P f = n! \int_P f.$$

展开严格证明 (Step 1-6 + coordinate-perm lemma)
Lemma (coordinate-permutation invariance)

对任意 $g \in \IR(\R^n)$ 与 $\sigma \in S_n$: $\int_{\R^n} g \circ T_\sigma = \int_{\R^n} g$.

证: $T_\sigma$ 是 linear 等距 (置换坐标), $\abs{\det T_\sigma} = 1$. 它把 dyadic cube $C_{\mathbf k, N}$ bijectively 映到另一 dyadic cube $C_{\mathbf k', N}$ (其中 $k'_j = k_{\sigma^{-1}(j)}$), 保 level + 保 vol. 故 $$U_N(g \circ T_\sigma) = \sum_C M_C(g \circ T_\sigma)\vol(C) = \sum_C M_{T_\sigma(C)}(g)\vol(C) \overset{C' = T_\sigma(C)}{=} \sum_{C'}M_{C'}(g)\vol(C') = U_N(g).$$ Similarly $L_N(g \circ T_\sigma) = L_N(g)$. $N \to \infty$ gives the integral equality. $\square$

证明 (a) 主体

Step 1 (几何): $Q = \bigcup_\sigma P_\sigma$ (每个 $x \in Q$ 至少存在排序 $\sigma$ 使 $x_{\sigma(1)} \le \cdots \le x_{\sigma(n)}$). 对 $\sigma \ne \tau$, $P_\sigma \cap P_\tau \subseteq B := \bigcup_{i < j}\{x_i = x_j\}$ (若 $x \in P_\sigma \cap P_\tau$ has all distinct coords, $\sigma$ 与 $\tau$ both = unique sort permutation, 矛盾).

Step 2 ($B$ has volume zero): 每个超平面 $H_{ij} = \{x_i = x_j\}$ 是 $(n-1)$-dim linear subspace, has measure zero (Prop 3.2.2). $B \cap Q$ 是 finite union of bounded portions ⟹ vol 0 (by finite additivity of vol on pavable measure-0 sets).

Step 3 (each $P_\sigma$ pavable): $\partial P_\sigma$ contained in finite union of $(n-1)$-dim polyhedral pieces ⟹ measure 0 ⟹ $\mathbf 1_{P_\sigma}$ continuous a.e. ⟹ pavable. 故 $\int_{P_\sigma} f := \int f \cdot \mathbf 1_{P_\sigma}$ well-defined.

Step 4 (almost-disjoint additivity): 设 $H := \sum_\sigma \mathbf 1_{P_\sigma} - \mathbf 1_Q$. 对 $x \notin Q$: $H(x) = 0$. 对 $x \in Q \setminus B$ (distinct coords): exactly one $\sigma$ orders, $\sum = 1 = \mathbf 1_Q$. 对 $x \in B \cap Q$: $0 \le H(x) \le n! - 1$. 综合 $0 \le H \le n! \cdot \mathbf 1_B$. 由 vol $B = 0$: $\int f \cdot H = 0$, 故 $$\int_Q f = \sum_\sigma \int_{P_\sigma} f.$$

Step 5 (对称性, 应用 Lemma): $f$ symmetric ⟹ $f \circ T_\sigma = f$ pointwise. 也 $\mathbf 1_{P_\sigma}(T_\sigma x) = \mathbf 1_P(x)$ (verify: $T_\sigma x \in P_\sigma \iff x_1 \le x_2 \le \cdots \le x_n \iff x \in P$). 故 $(f \cdot \mathbf 1_{P_\sigma}) \circ T_\sigma = f \cdot \mathbf 1_P$. By Lemma: $$\int_{P_\sigma} f = \int f \cdot \mathbf 1_{P_\sigma} = \int (f \cdot \mathbf 1_{P_\sigma}) \circ T_\sigma = \int f \cdot \mathbf 1_P = \int_P f.$$

Step 6 (合并): $$\int_Q f = \sum_\sigma \int_{P_\sigma} f = \sum_\sigma \int_P f = n! \int_P f. \qquad \blacksquare$$

Part (b) — $\int_P f(x_1)\cdots f(x_n) = \tfrac{1}{n!}(\int_a^b f)^n$

📋 Problem 4.3.4(b)

$f \colon [a, b] \to \R$ integrable. 证 $$\int_P f(x_1) f(x_2) \cdots f(x_n)\,\abs{d^n x} = \frac{1}{n!}\left(\int_a^b f\right)^n.$$

策略

设 $F(x) := \widetilde f(x_1) \widetilde f(x_2) \cdots \widetilde f(x_n)$ 其中 $\widetilde f$ 是 $f$ 的 zero-extended 版本. 三步:

  1. $F$ integrable + $\int_{\R^n} F = (\int_a^b f)^n$: 由 "baby Fubini" (Lec 2.2): product of integrable functions 在 $\R^k \times \R^m$ 上 integrable + 积分 = 积分 product.
  2. $F$ symmetric: 由乘法 commutativity, $\prod_i \widetilde f(x_{\sigma(i)}) = \prod_j \widetilde f(x_j)$.
  3. $F$ supported in $Q_{a, b}^n$: 若 $x_i \notin [a, b]$ 某 $i$, $\widetilde f(x_i) = 0$ ⟹ $F = 0$.

把 (a) 应用到 $F$: $\int_Q F = n! \int_P F$. LHS = $\int_{\R^n} F = (\int_a^b f)^n$, RHS = $n! \int_P F$. 整理得 $$\int_P F = \tfrac{1}{n!}(\int_a^b f)^n.$$

展开证明 (b)
证明

设 $\widetilde f(t) := f(t)$ for $t \in [a, b]$, $= 0$ otherwise. $\widetilde f$ bounded with bounded support $[a, b]$; $\Disc(\widetilde f) \subseteq \{a, b\} \cup \Disc(f)$ measure 0 ⟹ $\widetilde f \in \IR(\R)$. 设 $$F(x_1, \ldots, x_n) := \widetilde f(x_1) \widetilde f(x_2) \cdots \widetilde f(x_n).$$

(i) $F$ integrable on $\R^n$ + product formula: 由 baby Fubini (Lec 2.2): if $g_1 \in \IR(\R^k)$, $g_2 \in \IR(\R^m)$, then $g_1(x)g_2(y) \in \IR(\R^{k+m})$ with $\int g_1 g_2 = (\int g_1)(\int g_2)$. 归纳 $n - 1$ 次: $$\int_{\R^n} F = \prod_i \int_\R \widetilde f = \left(\int_a^b f\right)^n.$$

(ii) $F$ symmetric: $F(x_{\sigma(1)}, \ldots) = \prod_i \widetilde f(x_{\sigma(i)}) = \prod_j \widetilde f(x_j) = F(x)$ (real mul commutative).

(iii) $\supp F \subseteq Q^n_{a, b}$: 若 $x_i \notin [a, b]$ 某 $i$, $\widetilde f(x_i) = 0$ ⟹ $F(x) = 0$. 故 $\int_{\R^n} F = \int_{Q^n_{a, b}} F$.

应用 (a) 到 $F$: $F$ symmetric + integrable + supp $\subseteq Q$ ⟹ $$\left(\int_a^b f\right)^n = \int_{\R^n} F = \int_Q F = n! \int_P F = n! \int_P f(x_1)\cdots f(x_n)\,\abs{d^n x}.$$ 除以 $n!$: $$\int_P f(x_1) f(x_2) \cdots f(x_n)\,\abs{d^n x} = \frac{1}{n!}\left(\int_a^b f\right)^n. \qquad \blacksquare$$

💡 反思 — 排列群对称性: 几何 + 测度 + integral 三层

P 4.3.4 是 "对称性导致积分等式" 的典范, 三层结构相互锁定:

  1. 几何对称: $Q = \bigcup_\sigma P_\sigma$ + 对角集 $B$. 这是 cube 的 "symmetric group action 自由 (mod $B$)" 几何分解.
  2. 测度对称: $B$ measure 0, 故所有 $P_\sigma$ 在测度意义下 disjoint. 这让"$\int_Q = \sum_\sigma \int_{P_\sigma}$" 严格成立.
  3. 积分对称: coordinate-permutation invariance lemma + $f$ symmetric ⟹ $\int_{P_\sigma} f = \int_P f$ for all $\sigma$. 求和 ⟹ $n!$ factor.

三层缺一不可: 没几何分解, 求和无意义; 没测度对称, 求和有 overlap; 没积分对称, 每个 $\int_{P_\sigma}$ 不相等.

(b) 的 elegance: 把"对称性"免费得到 (real number multiplication commutative), 然后免费调用 (a). 这就让一个 $n$-维积分 — 看似复杂 — 简化为 1-D integral 的$n$ 次方 / $n!$. 公式形式直接对应 multinomial / Bose statistics: $\frac{1}{n!}$ 是"indistinguishable particles 修正因子"的初次出现.

与 Topic 03 (积分代数性质) 的关联: 那里讲 linearity / monotonicity / abs value 等基本性质, P 4.3.4 是对称性这一另一类 integral 性质 — 来自 group action 而非 algebra. 见 Topic 03.

§6 Problem 4.4.2 — Measure zero via balls (vs cubes)

📋 Problem 4.4.2

证: $X \subseteq \R^n$ has measure 0 ⟺ for any $\eps > 0$, exists infinite sequence of balls $B_i = \{x : \abs{x - a_i} < r_i\}$ with $\sum_i r_i^n < \eps$ and $X \subseteq \bigcup_i B_i$.

策略

题目要证 measure-zero 的两种定义等价: 原定义 (Def 3.1.1) 用 cubes; 这道题给的ball版本是 Hubbard 教材里用的 alternative. 两个定义形式不同但 morally 等价 — relating 它们需要一个几何 lemma:

🔧 Geometric Lemma (cube ↔ ball nesting)
  • (1) 开 cube $C(a, s)$ 边长 $s$ ⟹ $C(a, s) \subseteq B(a, s\sqrt n / 2)$.
  • (2) 开 ball $B(a, r)$ ⟹ $B(a, r) \subseteq C(a, 2r)$.

证 (1): $x \in C(a, s)$ ⟹ $\abs{x_i - a_i} < s/2$ each $i$ ⟹ $\abs{x - a}^2 = \sum (x_i - a_i)^2 < n (s/2)^2$ ⟹ $\abs{x - a} < s\sqrt n/2$.

证 (2): $x \in B(a, r)$ ⟹ $\abs{x_i - a_i} \le \abs{x - a} < r$ each $i$ ⟹ $x \in C(a, 2r)$.

有了 lemma, 双向都立即: cube cover with vol $< \eps_0$ 通过 lemma (1) 变 ball cover with $\sum r^n < (\sqrt n/2)^n \eps_0$ — 选 $\eps_0 = \eps / (\sqrt n/2)^n$ 让最终 $\sum r^n < \eps$. 反向: ball cover with $\sum r^n < \eps$ 通过 lemma (2) 变 cube cover with vol $< 2^n \eps$ — 选 $\eps = \eps' / 2^n$.

两个定义只差dimensional constants $2^n$ 与 $(\sqrt n/2)^n$, 这些常数被 $\eps$-budget吸收, 不影响"measure 0"的判定.

展开严格证明 (两 direction + lemma)
($\Rightarrow$) cube cover ⟹ ball cover

$X$ measure zero. 设 $\eps > 0$, $\eps_0 := \eps / (\sqrt n / 2)^n$. By Def 3.1.1, $\exists$ countable cube cover $\{C(a_i, s_i)\}$ of $X$ with $\sum s_i^n < \eps_0$. By Lemma (1), $C(a_i, s_i) \subseteq B(a_i, s_i \sqrt n / 2)$. 设 $r_i := s_i \sqrt n / 2$. $\{B(a_i, r_i)\}$ countable ball cover of $X$ with $$\sum_i r_i^n = (\sqrt n/2)^n \sum_i s_i^n < (\sqrt n/2)^n \eps_0 = \eps. \qquad \checkmark$$

($\Leftarrow$) ball cover ⟹ cube cover

设 $\eps' > 0$, $\eps := \eps' / 2^n$. By hypothesis, $\exists$ countable ball cover $\{B(a_i, r_i)\}$ of $X$ with $\sum r_i^n < \eps$. By Lemma (2), $B(a_i, r_i) \subseteq C(a_i, 2r_i)$ (cube of side $2r_i$). $\{C(a_i, 2r_i)\}$ countable cube cover with $$\sum_i \vol_n(C(a_i, 2r_i)) = \sum_i (2r_i)^n = 2^n \sum_i r_i^n < 2^n \eps = \eps'.$$ By Def 3.1.1, $X$ measure zero. $\checkmark$ $\blacksquare$

💡 反思 — 多种"测度零"定义的等价性

除了 cube + ball, 还可用 closed cubes / arbitrary boxes / $n$-dim 球 / ellipsoid 等定义 measure zero, 都等价. 一般原则: 任何 "shape" 满足 "volume $\asymp$ (linear size)$^n$" 的几何 family 都能 define measure zero, 因为可以通过$\asymp$ 常数互相 cover.

这与"measure 0 是一个 affine-invariant 概念" 一致 — 任何 invertible 线性变换 (= cube/ball 互相 morph) 都保 measure zero.

$r_i^n$ vs $c_n r_i^n$ (其中 $c_n = \pi^{n/2}/\Gamma(n/2 + 1)$ 是 $n$-球体积常数): 题目给的 $\sum r_i^n < \eps$ 没有 $c_n$ — 形式上"错" (因为 ball 体积应是 $c_n r^n$). 但实际是 conventionally 把 $c_n$ 吸收进 $\eps$ — 反正"$\sum c_n r_i^n < c_n \eps$" 等价于"$\sum r_i^n < \eps$" (任意 $\eps$). 这种"吸收 dimensional constant" 是 measure 理论的常用 maneuver. 见 Topic 05.

§7 Problem 4.4.7 — Set of numbers using digits 1-6 (in base 10) has measure 0

📋 Problem 4.4.7

证: $S := \{x \in \R : x$ has base-$10$ expansion using only digits in $\{1, 2, 3, 4, 5, 6\}\}$ has measure 0.

策略

这是Cantor-like构造的标准 measure-0 检查: 把 $S$ 按数字 prefix 长度 $m$ 枚举, 用越来越 fine 的 cover 让总长趋 0.

Step 1 (reduce 到 bounded slabs): $S = \bigcup_{k \in \Z}(S \cap [k, k+1])$ countable union. 由 Thm 3.1.1, 每个 $S_k = S \cap [k, k+1]$ measure 0 ⟹ $S$ measure 0.

Step 2 (每个 $S_k$ measure 0 via prefix enumeration): $x \in S_k \cap (k, k+1)$ has expansion $x = k.d_1 d_2 \cdots$ with $d_i \in \{1, \ldots, 6\}$. 对固定 prefix length $m$, 共 $6^m$ possible prefixes; 每个 prefix 对应一个长 $10^{-m}$ 的 closed 区间 $I_{\mathbf d}$. $S_k$ 覆盖于 $6^m$ 个这样的区间 (加 endpoints $\{k, k+1\}$ measure 0).

关键的 rate: $$L_m := 6^m \cdot 10^{-m} = (3/5)^m \;\xrightarrow{m \to \infty}\; 0.$$ 因 $6 < 10$, ratio $6/10 = 3/5 < 1$, 几何序列趋 0. 对任意 $\eps$, 选 $m$ 足够大让 $L_m < \eps/2$, 再 fatten 每个 $I_{\mathbf d}$ 一点点变开 cover, 总和 $< \eps$.

展开严格证明
证明

Step 1 (reduction): $S = \bigcup_{k \in \Z}S_k$ where $S_k := S \cap [k, k+1]$. By Thm 3.1.1, suffices to show each $S_k$ measure 0.

Step 2 (prefix covers): $x \in S_k \cap (k, k+1)$: $x = k.d_1 d_2 \cdots$, $d_i \in \{1, \ldots, 6\}$. 对 $m \in \N$, 给固定 prefix $\mathbf d = (d_1, \ldots, d_m) \in \{1, \ldots, 6\}^m$, $x$ with prefix $\mathbf d$ lies in $$I_{\mathbf d} := \left[k + \sum_{j=1}^m d_j / 10^j,\;\; k + \sum_{j=1}^m d_j / 10^j + 1/10^m\right] \text{ (length } 10^{-m}\text{)}.$$ 故 $S_k \subseteq \{k, k+1\} \cup \bigcup_{\mathbf d} I_{\mathbf d}$ (有限 union of $6^m$ 闭区间).

Total length $L_m = 6^m \cdot 10^{-m} = (3/5)^m \to 0$.

Step 3 (开 cover): 对 $\eps > 0$, 选 $m$ 使 $L_m < \eps/2$. 把每 $I_{\mathbf d}$ fatten 到开区间 $J_{\mathbf d}$ of length $10^{-m} + 2\eta$ where $\eta := \eps/(8 \cdot 6^m)$. Endpoints $k, k+1$ 各覆盖 open 区间 of length $\eta$. 总长: $$6^m \cdot (10^{-m} + 2\eta) + 2\eta = L_m + 2\eta(6^m + 1) < \eps/2 + 2 \cdot \frac{\eps}{8 \cdot 6^m} \cdot (6^m + 1) < \eps/2 + \eps/2 = \eps.$$

开 cover 是 finite, 但 Def 3.1.1 要 countable cover — pad with empty 开区间 (or shrinking 开区间) 让 cover countable, 总长仍 $< \eps$. 故 $S_k$ measure 0. $\blacksquare$

💡 反思 — Cantor-like 构造的 generic 模板

更一般: 对任意真子集 $D \subsetneq \{0, 1, \ldots, 9\}$, set of numbers using only digits in $D$ has measure 0, 由覆盖率 $(\abs D / 10)^m \to 0$. 这给出 大量 测度零集 — 与最经典的 Cantor 三分集同型 (那里 $D = \{0, 2\}$ in base 3, ratio $(2/3)^m \to 0$).

Cantor-like measure-0 集的uncountable性: 这些集合大 (与 $D^\N$ 一一对应, uncountable, in fact cardinality $\mathfrak c = $ continuum) 但 measure 0. 这是与 P3 (countable ⟹ measure 0) 形成巨大对比: measure 0 不蕴含 countable. 一个集合可以"uncountably large but volumetrically negligible".

这种strange集合是Lebesgue 理论 mass-energy gap 的反例库: 例如 Cantor 集 measure 0 yet uncountable + perfect (closed + 无 isolated points) + 与 $[0, 1]$ 等势. 是 measure 理论主要 motivation. 见 Royden Ch 2.

§8 Problem 4.5.8 — Cylinder slice 体积 via Fubini

📋 Problem 4.5.8

计算 cylinder slice 体积 of $W = \{(x, y, z) : x^2 + y^2 \le 1,\;-1/2 \le y \le 1/2,\;0 \le z \le 2\}$ via iterated integral.

策略

$W$ 是 unit cylinder (axis = $z$-axis) 被 $y = \pm 1/2$ 切的 slab, 再 truncated 到 $z \in [0, 2]$. 直接 Fubini 即可: 投影到 $y$-axis $\to y \in [-1/2, 1/2]$, fixed $y$ 的 cross-section is rectangle in $(x, z)$ plane with $x \in [-\sqrt{1 - y^2}, \sqrt{1 - y^2}]$ + $z \in [0, 2]$. 故 $$\vol_3(W) = \int_{-1/2}^{1/2}\int_{-\sqrt{1-y^2}}^{\sqrt{1-y^2}}\int_0^2 1\,dz\,dx\,dy.$$ 内积分 trivial: $\int_0^2 dz = 2$, $\int_{-\sqrt{1-y^2}}^{\sqrt{1-y^2}} 2 dx = 4\sqrt{1 - y^2}$. 外 1-D: $\int_{-1/2}^{1/2} 4\sqrt{1 - y^2} dy$ — 用 $y = \sin\theta$, $\theta \in [-\pi/6, \pi/6]$.

展开计算
计算

$W$ pavable: $\partial W \subseteq$ finite union of smooth surfaces (cylinder, planes) — measure 0. By Fubini: $$\vol_3(W) = \int_{-1/2}^{1/2}\int_{-\sqrt{1-y^2}}^{\sqrt{1-y^2}}\int_0^2 1\,dz\,dx\,dy = \int_{-1/2}^{1/2} 4\sqrt{1 - y^2}\,dy.$$

代换 $y = \sin\theta$, $dy = \cos\theta d\theta$, $\theta \in [-\pi/6, \pi/6]$ (因 $\sin(\pm \pi/6) = \pm 1/2$). $\sqrt{1 - y^2} = \cos\theta$: $$\vol_3(W) = 4 \int_{-\pi/6}^{\pi/6}\cos^2\theta\,d\theta = 4 \int_{-\pi/6}^{\pi/6}\frac{1 + \cos 2\theta}{2}\,d\theta = 2\left[\theta + \tfrac{1}{2}\sin 2\theta\right]_{-\pi/6}^{\pi/6}.$$

$\theta = \pi/6$: $\pi/6 + \sin(\pi/3)/2 = \pi/6 + \sqrt 3/4$. 由 odd, $\theta = -\pi/6$ 给负值. 差 $= 2(\pi/6 + \sqrt 3/4) = \pi/3 + \sqrt 3/2$. 故 $$\boxed{\;\vol_3(W) = 2(\pi/3 + \sqrt 3/2) = 2\pi/3 + \sqrt 3 \approx 3.826.\;} \qquad \blacksquare$$

💡 反思 — 几何 vs 代数 vs 答案 一致性

$W$ 是 unit cylinder ($\vol = \pi \cdot 1^2 \cdot 2 = 2\pi$, 大约 6.28) 被 $\abs y \le 1/2$ 切下的中央 slab. 这个 slab 占 cylinder 的多大比例?

cylinder 横截面 (disk of radius 1) 被 $\abs y \le 1/2$ 切下的中央带 (rectangle 加两个 circular caps). 这个带的面积 = disk area $\pi$ - 两个 "side caps" 面积. 每个 cap = $\int_{1/2}^1 2\sqrt{1 - y^2} dy$. 算: $y = \sin\theta$, $\int_{\pi/6}^{\pi/2} 2\cos^2\theta d\theta = [\theta + \sin 2\theta / 2]_{\pi/6}^{\pi/2} = (\pi/2 + 0) - (\pi/6 + \sqrt 3 /4) = \pi/3 - \sqrt 3/4$. 两 caps 总 = $2\pi/3 - \sqrt 3/2$.

带面积 = $\pi - (2\pi/3 - \sqrt 3/2) = \pi/3 + \sqrt 3/2$. 乘 cylinder 高 $z \in [0, 2]$ length 2: 体积 = $2 \pi/3 + \sqrt 3 \approx 3.826$. $\checkmark$ — 与上面 Fubini 答案一致.

验证比例: $(2\pi/3 + \sqrt 3) / (2\pi) \approx 3.826 / 6.28 \approx 60.9\%$ — 大约 60% 的 cylinder 在 $\abs y \le 1/2$ slab 里, 合理 (因为 $y = \pm 1/2$ 切掉 disk 上下 $\sim 18\%$ each).

§9 Problem 4.5.11 — ★ Swap-order trick on $\int_0^\pi \int_y^\pi \sin x/x\,dx\,dy$

📋 Problem 4.5.11

(a) 把 $\int_0^\pi(\int_y^\pi \sin x / x\,dx)\,dy$ 表达为 $\R^2$ 上某区域的积分.
(b) 用 other order of iteration.
(c) Compute.

策略 — ★ 经典 swap-order trick

这是 Fubini 最经典的非平凡应用: $\int \sin x / x\,dx$ 没有 elementary antiderivative (sine integral $\mathrm{Si}(x)$ 是 special function), 所以原顺序 内积分 $\int_y^\pi \sin x / x dx$ 不能 closed-form 求出. 但 互换顺序 后, 内积分变成 $\int_0^x \sin x / x dy = \sin x / x \cdot x = \sin x$ — 完全 elementary!

Step 1 (区域): 原 iterated 区域 $R = \{(x, y) : 0 \le y \le \pi,\;y \le x \le \pi\} = \{0 \le y \le x \le \pi\}$. 这是三角形 with vertices $(0, 0), (\pi, 0), (\pi, \pi)$.

Step 2 (other order): 投影到 $x$-axis: $x \in [0, \pi]$, fixed $x$: $y \in [0, x]$. 故 $$\int_R g = \int_0^\pi \int_0^x \frac{\sin x}{x} dy\,dx.$$

Step 3 (compute): 内: $\int_0^x \sin x / x dy = \sin x$ (因 $\sin x / x$ 与 $y$ 无关). 外: $\int_0^\pi \sin x dx = [-\cos x]_0^\pi = 1 - (-1) = 2$. 答案 = 2.

展开严格证明 (含 integrability check + extend)
证明

(a) 区域 + integrability: $R = \{0 \le y \le x \le \pi\}$ — 三角形 with vertices $(0, 0), (\pi, 0), (\pi, \pi)$. 整数函数 $\sin x / x$ 在 $x = 0$ undefined, but extends continuously by $\lim_{x \to 0}\sin x / x = 1$ (Taylor 展开). 设 $$g(x, y) := \begin{cases}\sin x / x & x \ne 0 \\ 1 & x = 0\end{cases}$$ on $R$. $g$ continuous on compact $R$, 故 $g \in \IR$ (by P2 / Lebesgue 准则), 且 $$\int_0^\pi \int_y^\pi \frac{\sin x}{x}\,dx\,dy = \int_R g\,\abs{d^2 z}.$$

(b) other order: 投影 $R$ 到 $x$-axis: $x \in [0, \pi]$, slice $\{y : 0 \le y \le x\}$. By Fubini: $$\int_R g\,\abs{d^2 z} = \int_0^\pi \int_0^x \frac{\sin x}{x}\,dy\,dx.$$

(c) compute: 内 ($\sin x / x$ const in $y$): $\int_0^x \sin x / x\,dy = (\sin x / x) \cdot x = \sin x$ (for $x > 0$; $x = 0$ degenerate slice 贡献 0). 外: $$\int_R g\,\abs{d^2 z} = \int_0^\pi \sin x\,dx = [-\cos x]_0^\pi = -(-1) - (-1) = 2.$$ $$\boxed{\;\int_0^\pi\int_y^\pi \frac{\sin x}{x}\,dx\,dy = 2.\;} \qquad \blacksquare$$

💡 反思 — Fubini 的"顺序 != 都好做"

Fubini 不仅 swap 顺序使两 iterated integrals 都给同一答案, 还选最方便的计算路径. 这道题的 punchline: 原顺序 $\int_y^\pi \sin x / x dx$ 是 "$\mathrm{Si}(\pi) - \mathrm{Si}(y)$" — special function, 不能 elementary; 但 swap 后内积分立刻成 trivial. 这就是 "Fubini 救场" 模板.

更深的事实: 原积分实际是 $\int_0^\pi \mathrm{Si}(\pi) - \mathrm{Si}(y) dy = \pi \mathrm{Si}(\pi) - \int_0^\pi \mathrm{Si}(y) dy$. $\mathrm{Si}(\pi) \approx 1.85$. 直接算这两项加起来要复杂 IBP. 但 swap 顺序后立得 2 (与 $\pi\mathrm{Si}(\pi) - \int \mathrm{Si}$ 数值吻合).

实战教训: 看到 "iterated integral with non-elementary integrand" 第一反应 — try swapping order. 80% 的情况下 swap 后有简化, 因为 "$f$ 与 inner variable 无关" 模式 (这里 $\sin x / x$ inner $dy$ 时常数) 是最常见.

对比 Topic 07 (Fubini): Topic 07 给出 Fubini 主定理 + 主要应用 (Steinmetz, ball volume, etc.) — P 4.5.11 是 swap-order trick 的标志性 worked example. 互相补全. 见 Topic 07.

§10 Problem 4.5.15 — Paraboloids 间体积

📋 Problem 4.5.15

找 region $W = \{(x, y, z) : z \ge x^2 + y^2,\; z \le 10 - x^2 - y^2\}$ 的体积.

策略

$W$ 是两个 paraboloid 之间的 region: 底是 $z = x^2 + y^2$ (开口向上), 顶是 $z = 10 - x^2 - y^2$ (开口向下). 投影到 $xy$-plane: $(x, y)$ 必满足 $x^2 + y^2 \le 10 - x^2 - y^2 \iff x^2 + y^2 \le 5$ — 半径 $\sqrt 5$ 圆盘 $D$.

By Fubini, 先积 $z$: $$\vol_3(W) = \int_D (10 - 2(x^2 + y^2))\,dV_2.$$ 然后用 Cartesian $y$-slicing + Wallis 积分 (因课程暂不允许 polar) 算外积分.

关键步骤: 内积分 ($y$ over $[-\sqrt{5 - x^2}, \sqrt{5 - x^2}]$) 给 $\tfrac{8}{3}(5 - x^2)^{3/2}$ (经 algebraic 化简). 外 $x$-积分用 $x = \sqrt 5 \sin\theta$ 代换变成 $25 \int_0^{\pi/2} \cos^4\theta d\theta$ — 经典 Wallis 积分 $= 3\pi/16$. 最终 $\vol_3 = 25\pi$.

展开计算
计算

投影确定: $W$ nonempty 处, $x^2 + y^2 \le z \le 10 - x^2 - y^2$ ⟹ $2(x^2 + y^2) \le 10 \iff x^2 + y^2 \le 5$. 故 $W$ 投影到 $xy$-plane 是 $D = \{(x, y) : x^2 + y^2 \le 5\}$.

Fubini: 边界 finite union of smooth surfaces ⟹ pavable. 先积 $z$: $$\vol_3(W) = \int_D\int_{x^2+y^2}^{10-x^2-y^2} 1\,dz\,dV_2 = \int_D(10 - 2(x^2 + y^2))\,dV_2.$$

$y$-slicing: 投影 $D$ to $x$-axis: $x \in [-\sqrt 5, \sqrt 5]$; fixed $x$: $y \in [-\sqrt{5 - x^2}, \sqrt{5 - x^2}]$. 设 $A := 10 - 2x^2$, $u := \sqrt{5 - x^2}$, $A = 2u^2$: $$\int_{-u}^u (A - 2y^2)\,dy = 2Au - 2 \cdot \frac{2}{3}u^3 = 4u^3 - \frac{4}{3}u^3 = \frac{8}{3}u^3 = \frac{8}{3}(5 - x^2)^{3/2}.$$

故 $$\vol_3(W) = \int_{-\sqrt 5}^{\sqrt 5}\frac{8}{3}(5 - x^2)^{3/2}\,dx = \frac{16}{3}\int_0^{\sqrt 5}(5 - x^2)^{3/2}\,dx \quad\text{(by evenness)}.$$

三角代换 $x = \sqrt 5 \sin\theta$: $dx = \sqrt 5 \cos\theta d\theta$, $\theta \in [0, \pi/2]$, $(5 - x^2)^{3/2} = 5^{3/2}\cos^3\theta$: $$\int_0^{\sqrt 5}(5 - x^2)^{3/2}\,dx = \int_0^{\pi/2} 5^{3/2}\cos^3\theta \cdot \sqrt 5\cos\theta\,d\theta = 25 \int_0^{\pi/2}\cos^4\theta\,d\theta.$$

Wallis 积分: $\int_0^{\pi/2}\cos^4\theta d\theta = 3\pi/16$ (用 $\cos^4 = \tfrac{3}{8} + \tfrac{1}{2}\cos 2\theta + \tfrac{1}{8}\cos 4\theta$, 后两项 $\int_0^{\pi/2} = 0$). 故 $$\int_0^{\sqrt 5}(5-x^2)^{3/2}\,dx = 25 \cdot \frac{3\pi}{16} = \frac{75\pi}{16}.$$

合并: $$\vol_3(W) = \frac{16}{3} \cdot \frac{75\pi}{16} = \frac{75\pi}{3} = 25\pi. \qquad \boxed{\;\vol_3(W) = 25\pi \approx 78.54.\;}\qquad \blacksquare$$

💡 反思 — Cartesian vs Polar (Lec 6.1 预告)

用polar coordinates (尚未学到, deferred 到 Lec 5.3+) 这道题一行就完: $D$ 是 disk radius $\sqrt 5$, 用 $r \in [0, \sqrt 5]$, $\theta \in [0, 2\pi]$, Jacobian $r$: $$\vol_3 = \int_0^{2\pi}\int_0^{\sqrt 5}(10 - 2r^2)\,r\,dr\,d\theta = 2\pi\int_0^{\sqrt 5}(10r - 2r^3)\,dr = 2\pi[5r^2 - r^4/2]_0^{\sqrt 5} = 2\pi(25 - 25/2) = 2\pi \cdot 25/2 = 25\pi.$$ 极坐标自然匹配 axial symmetry (paraboloid 都 radial), 把"非平凡"$3D$ 积分变 trivial $1D$.

但本周不允许 polar: HW2 在 Lec 4.1-4.2 阶段, polar coord (Lec 5.3 主题, Topic 09 nonlinear CoV) 尚未引入. 故必须用 Cartesian + Wallis 积分 — 等价但 algebra 长.

三阶段计算 difficulty 对比:

  • Cartesian + Fubini + Wallis (本题): ~10 行 algebra, 4 步代换, 用 special integral.
  • Cylindrical coords (Lec 5.3, Topic 09): ~2 行, $r dr$ Jacobian.
  • Spherical coords: 不 natural for paraboloid (没 spherical symmetry); 反而麻烦.

这道题预告了"选对坐标系能化解 90% 的复杂度" — 整 Ch 5-6 的核心 mantra. 见 Topic 10 (CoV 三大设计模式) 把 "对称性驱动" 列为第一模式 — paraboloid 的 axial symmetry 正符合.

§11 方法论速查

  • Measure zero (Def 3.1.1): countable cube cover with total vol $< \eps$ for all $\eps > 0$. P1, P3, P4 全, P 4.4.2/7 全.
  • Volume zero: $\mathbf 1_X$ integrable with $\int \mathbf 1_X = 0$. 严格强于 measure zero (但相反不成立).
  • Undefined volume: $\mathbf 1_X$ not integrable. $\Q \cap [0, 1]$ 标志例.
  • $U_N(\mathbf 1_X) = \sum_{C \cap X \ne \emptyset}\vol_n(C)$: indicator 的 upper sum = "touches $X$" 的 cube 总体积. P1 reverse direction 工具.
  • Thomae's function: $f(p/q) = 1/q$ (lowest terms), $f(\text{irrational}) = 0$. $\Disc f = \Q$. P4.
  • Symmetric integrable function: $f(x_{\sigma(1)}, \ldots) = f(x_1, \ldots)$ for all $\sigma \in S_n$. P 4.3.4.
  • Ascending simplex $P^n_{a, b}$: $\{a \le x_1 \le \cdots \le x_n \le b\} \subset Q^n_{a, b}$. $\vol(P) = (b-a)^n / n!$, 恰为 cube 的 $1/n!$.
  • ★ $\eps/2^j$-trick (P3): 对 countable family 用 $\eps/2^{j+1}$ 体积预算, 总和 $\le \eps/2$. countable measure-zero 的基本技术.
  • Countable ⟹ measure 0 (P3): 任意可数集 $S \subseteq \R^n$ has measure 0.
  • Lebesgue 准则 (Thm 4.1.13): $f$ 可积 ⟺ $\Disc f$ measure 0. P4 Thomae 直接调用.
  • ★ $f$ continuous + bounded support ⟹ integrable (P2): 用 uniform continuity + dyadic diameter $< \min(\delta, 1)$, case-by-case $C \subseteq K$ vs $C \not\subseteq K$.
  • ★ Coordinate-permutation invariance (P 4.3.4): $\int g \circ T_\sigma = \int g$ for $\sigma \in S_n$. 由 $T_\sigma$ permutes dyadic cubes + 保 vol.
  • Symmetric integral on cube vs simplex (P 4.3.4a): $\int_Q f = n! \int_P f$ for symmetric integrable $f$.
  • Product integral on simplex (P 4.3.4b): $\int_P f(x_1)\cdots f(x_n)\,\abs{d^n x} = \tfrac{1}{n!}(\int_a^b f)^n$.
  • Measure zero via balls (P 4.4.2): equivalent to cube definition; 几何 lemma $C(a, s) \subset B(a, s\sqrt n/2)$ + $B(a, r) \subset C(a, 2r)$.
  • Fubini (Thm 4.5.1): $f$ integrable on $\R^{n+m}$ ⟹ iterated integral存在并等于 multiple integral. P 4.5.8, 4.5.11, 4.5.15 全.
  • ★ Density + Cauchy 准则反向 (P1, P4): $X$ dense + complement dense in $I$ ⟹ $U_N(\mathbf 1_X) \ge \vol(I)$, $L_N = 0$ ⟹ $\mathbf 1_X \notin \IR$.
  • Dyadic diameter 估计: $\diam(C_{\mathbf k, N}) = \sqrt n / 2^N \to 0$. P2 直接用.
  • Fattening $\supp$: $K = [-R-1, R+1]^n$: 给 $\supp \subseteq [-R, R]^n$ 留 "$1$ 单位 safety margin", 让 $C \not\subseteq K$ ⟹ $f \equiv 0$ on $C$. P2 设计要点.
  • ★ Swap-order trick (P 4.5.11): 看到 non-elementary inner integral, swap iteration order, 经常 inner 变 trivial. $\sin x / x$ 是标志例.
  • 三角代换 $y = a\sin\theta$ for $\sqrt{a^2 - y^2}$: 把 $\sqrt{a^2 - y^2}$ 变 $a\cos\theta$, 让积分 elementary. P 4.5.8, P 4.5.15 同款.
  • Wallis 积分: $\int_0^{\pi/2}\cos^k\theta d\theta$ 公式 (k=2: π/4, k=4: 3π/16, k=6: 5π/32). P 4.5.15 用 $k = 4$.
  • Cantor-like construction (P 4.4.7): 数字 prefix 枚举 + 覆盖率 $(\abs D / \text{base})^m \to 0$. 给出大量 uncountable measure-0 集.
  • ★ measure 0 vs vol 0 vs undefined vol: vol 0 ⟹ measure 0 (永远), but reverse 不成立 ($\Q \cap [0, 1]$ 反例); undefined vol ⟹ not measure 0? 错! $\Q \cap [0, 1]$ measure 0 yet undefined vol — 三者是 distinct concepts.
  • Pitfall — countable ⟹ measure 0 但反向不成立: Cantor-like 集合 (P 4.4.7) measure 0 yet uncountable. "测度小"与"基数小"不同概念.
  • Pitfall — Fubini hypothesis 要 $f \in \IR(\R^{n+m})$: $f$ integrable on full space, not just sliced. 若仅 iterated integral exist 不蕴含 multiple integral exist (HW4 P 4.11.8 反例: $x^2 - y^2 / (x^2 + y^2)^2$).
  • Pitfall — Polar 在 HW2 不可用: P 4.5.15 看起来"应该" polar, 但 nonlinear CoV (Lec 5.3, Topic 09) 还没学. 必须用 Cartesian + Wallis. 不要"预判用 polar" 失分.
