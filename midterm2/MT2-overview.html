MT2 总览 + 综合速查

H&H §4.9-4.11 综合 · MT2 复习专区入口 · 考前 10 分钟 cheat + mock mini-final

导览

本 section 是为 Midterm 2 量身打造的复习入口, 完全针对 H&H §4.9-4.11 的 scope. 跟 Topics 01-14 的主线 (按 lecture 切分, 深度优先) 互为补充: MT2 这三页按课本章节切分, 内部按题型聚簇组织, 目标是"考前一周到一天"的 exam-prep 而非"长期建立 mental model".

三大主题, 一句话总结: §4.9 把 $\det T$ 从代数对象升级成"$T$ 拉伸体积的因子" — DSV (determinant scales volume) 与 linear CoV 都是这条几何观察的直接 corollary. §4.10 把常数 $\abs{\det T}$ 替换成逐点 Jacobian $\abs{\det[D\Phi(x)]}$, 得到非线性 change of variables — 极坐标 / 球坐标 / 柱坐标三大经典 + symmetry / boundary / slice-driven 三种 design pattern. §4.11 跳出 Riemann 框架的两大失败 (Dirichlet 太不连续 + $-\ln x$ 太无界), 用 Daniell-Stone 砖块分解构造 Lebesgue 积分, 证 well-definedness, 并立常用初等性质 (linearity / product with bounded R / monotonicity).

推荐使用法: (1) 考前 1 周, 按 MT2-A → MT2-B → MT2-C 顺序 deep-read 一遍, 每页 60-90 min, 包括 §3 题型聚簇与 §6 模拟练习. (2) 考前 1 天, 只扫各页 §1 速查 + §4 陷阱 + 本页 §3 跨章节速查 — 这是经过浓缩的"essential layer". (3) 考前 10 分钟, 扫本页 §3 速查 + §4 mock mini-final 的题目陈述 (不看答案, 检查自己能否快速 identify 解法).

跨章节速查表 — 10-min 扫读用

从 MT2-A/B/C §1 各页速查表中浓缩出的 essential layer. 若要细节请跳到对应主题页. 这版没有 cross-link 跳转, 设计为可打印 / iPad 上滚动一遍.

  • §4.9 DSV: $T$ 线性, $A$ pavable $\Rightarrow$ $\vol_n(T(A)) = \abs{\det T}\cdot\vol_n(A)$. 5-step 证: 构造 $T(\mathcal{D}_N)$ paving → nested → cube 同标度 → 推广到 $A$ → 用 elementary matrix 算 $\vol_n(T(Q_n)) = \abs{\det T}$.
  • §4.9 Linear CoV: $T$ 可逆线性, $f \in \IR(\R^n) \Rightarrow \int f\,dV = \abs{\det T}\int (f\circ T)\,dV$. 3-step 证: $T(\mathcal{D}_N)$ 上定义 $U_N^T$ → 用双射+DSV 转移 → 取极限.
  • §4.9 Orthogonal preserves vol: $S \in O(n) \Rightarrow \abs{\det S} = 1 \Rightarrow \vol(S(A)) = \vol(A)$. 加平移 = rigid motion 保体积.
  • §4.10 Nonlinear CoV (H&H Thm 4.10.12): $X$ 紧, $\vol_n(\partial X) = 0$, $\Phi \in C^1$ 在 $X\setminus\partial X$ 单射, $[D\Phi]$ 可逆 Lipschitz $\Rightarrow \int_{\Phi(X)} f\, dV = \int_X (f\circ\Phi)\,\abs{\det[D\Phi]}\, dV$.
  • §4.10 三大 Jacobian: polar $\abs{\det[DP]} = r$ (so $dV = r\,dr\,d\theta$). 球 (latitude $\varphi$) $\abs{\det[DS]} = r^2\cos\varphi$ ($z = r\sin\varphi$). 柱 $\abs{\det[DC]} = r$ (block-diagonal).
  • §4.11 Uniform Conv → swap: $(f_k) \subset \IR$, 公共 bounded supp, $f_k \xrightarrow{uni} f$ ⟹ $f \in \IR$ 白送 + $\lim \int f_k = \int f$.
  • §4.11 DCT for Riemann: 同 H1 + $|f_k| < M$ + a.e. $f_k \to f$ + $f \in \IR$ 付费 hypothesis ⟹ $\int f_k \to \int f$.
  • §4.11 Beppo-Levi: $\sum_k \int|f_k|\, dV < \infty$ ⟹ $\sum_k f_k$ a.e. 收敛. Chebyshev 论证.
  • §4.11 Lebesgue 定义: $f \in \IL$ iff $\exists (f_k) \subset \IR$ with $\sum\int|f_k| < \infty$ and $\sum f_k =_L f$; 定义 $\int f\, dV_L := \sum \int f_k$.
  • §4.11 良定性: 不依赖砖块选择 (truncation $[H_l]_R$ + Cauchy tail + Markov + DCT, 总账 $|\int H_l| < 6\eps$).
  • §4.11 初等性质: (a) 线性; (b) $f \in \IL,\ g \in \IR$ bounded $\Rightarrow fg \in \IL$; (c) 单调; (d) 三角. 不含一般乘积.
  • §4.9 三类 elementary matrix: Type 1 (scaling $m$) → $\vol(E(Q_n)) = \abs{m}$. Type 2 (shear) → $\vol = 1$ (Cavalieri). Type 3 (swap) → $\vol = 1$ ($Q_n$ 顶点重排).
  • §4.9 Multiplicativity: $\vol_n((ST)(Q_n)) = \vol_n(S(Q_n))\vol_n(T(Q_n))$. DSV 应用于 $A = T(Q_n)$ 得.
  • §4.9 经典 reduction: 椭圆 = $\mathrm{diag}(a,b)$(unit disk); 椭球 = $\mathrm{diag}(a,b,c)$(unit ball); simplex 体积 = $\abs{\det[\text{顶点}]}/n!$; parallelepiped = $\abs{\det[\text{edges}]}$.
  • §4.10 Pattern 1 Symmetry-driven: 区域/被积函数对称 → 选对应坐标. 难点在积分顺序而非选哪个坐标系.
  • §4.10 Pattern 2 Boundary-driven: 边界 $\{g_i = c_i\}$ → $\Psi = (g_1,\ldots,g_n)$ 把区域映成 axis-aligned box; 用 $\Phi = \Psi^{-1}$. IFT shortcut: $\det[D\Phi](\Psi(x)) = 1/\det[D\Psi](x)$ — 含 fractional 时不要微分 $\Phi$.
  • §4.10 Pattern 3 Slice-driven: $z$-切片是 parametric 形状 → $\Phi = (\text{slice param}, z)$, 末行 $(0,0,1)$ 折叠 $3\times 3$ det 到左上 $2\times 2$.
  • §4.10 单射验证: explicit inverse 通常最直接. 角度变量 $\theta \in [0, 2\pi)$ 半开避免覆盖两次.
  • §4.11 三个 reference 砖块: (1) Dirichlet $\mathbf{1}_{\Q\cap[0,1]}$ via $\mathbf{1}_{\{a_k\}}$, $\int_L = 0$. (2) $-\ln x \cdot \mathbf{1}_{(0,1]}$ via dyadic intervals, $\int_L = 1$. (3) $\mathbf{1}_{[-1,1]}/\sqrt{|x|}$, $\int_L = 4$ via 几何级数.
  • §4.11 三个反例 (mass loss): traveling box $\mathbf{1}_{[k,k+1]}$ (逃 $\infty$); tall thin spike $k\,\mathbf{1}_{(0,1/k]}$ (集中 0); cumulative Dirichlet $\mathbf{1}_{\{a_1,\ldots,a_k\}}$ ($\int\lim$ 在 R 下无定义). 都满足 $\int f_k = 1$ 但 lim = 0.
  • 陷阱: 必须用 $\abs{\det}$ — 体积非负, $\det$ 可负. 适用 §4.9 与 §4.10. (Ch 5-6 forms 才用有符号 det.)
  • 陷阱 §4.9 CoV 方向: $\int f(y)\,dV = \abs{\det T}\int f(T(x))\,dV$ — 因子贴在代换后那一侧 (与 1D $g'(x)$ 同侧).
  • 陷阱 $\abs{\det T} = 1 \ne T \in O(n)$: shear 反例 — 保面积但不保长度.
  • 陷阱 §4.10 latitude vs colatitude: H&H 用 latitude $\varphi \in (-\pi/2, \pi/2)$, Jacobian $r^2\cos\varphi$. 看 $\cos$ 或 $\sin$ 反推约定.
  • 陷阱 §4.10 忘单射: 极坐标 $\theta$ 半开避免覆盖; 球坐标极点退化 OK 因为边界测度零.
  • 陷阱 §4.10 IFT shortcut 倒: $\det[D\Phi] = 1/\det[D\Psi]$ — 不要算反取反.
  • 陷阱 §4.11 DCT 与 Uniform conv 区别: Uniform 送 $f \in \IR$; DCT 要求 $f \in \IR$ upfront — 不要混.
  • 陷阱 §4.11 绝对求和: 是 $\sum \int |f_k| < \infty$ (绝对值在积分内), 不是 $\sum |\int f_k| < \infty$.
  • 陷阱 §4.11 product 不封闭: $f, g \in \IL \not\Rightarrow fg \in \IL$. 反例 $f = 1/\sqrt{|x|}\cdot\mathbf{1}_{[-1,1]}$, $f^2 = 1/|x|\cdot\mathbf{1}_{[-1,1]} \notin \IL$ (log 发散).
  • 陷阱: "Lebesgue 准则" ≠ "Lebesgue 积分" — Topic 06 vs §4.11. 前者判 $\IR$, 后者是新 framework.

Mock Mini-Final (5 题 · 跨 §4.9-4.11 综合)

难度: 接近 MT2 实际试题 (估 ~30% 题量). 展开 <details> 查看答案. 建议先自己做 45 min 再看.

1. (§4.9 主题) 设 $\Omega \subset \R^3$ 是四面体, 顶点 $\mathbf{0}, v_1 = (1, 2, 0), v_2 = (3, 0, 1), v_3 = (0, -1, 4)$. 求 $\vol_3(\Omega)$.

答案

设标准 simplex $T_1$ 顶点 $\mathbf{0}, e_1, e_2, e_3$, $\vol_3(T_1) = 1/6$ (Fubini 给).

构造 $S \colon \R^3 \to \R^3$ 列向量 = $v_1, v_2, v_3$: $$[S] = \begin{pmatrix} 1 & 3 & 0 \\ 2 & 0 & -1 \\ 0 & 1 & 4 \end{pmatrix}.$$ 由线性映射保凸包, $S(T_1) = \Omega$. 沿第 1 列展开 $\det S$: $$\det S = 1\cdot(0\cdot 4 - (-1)\cdot 1) - 2\cdot(3\cdot 4 - 0\cdot 1) + 0 = 1\cdot 1 - 2\cdot 12 = 1 - 24 = -23.$$ $\abs{\det S} = 23$. 由 DSV: $$\vol_3(\Omega) = \abs{\det S}\cdot \vol_3(T_1) = 23 \cdot (1/6) = \boxed{23/6}.$$

陷阱检查: 用 $\abs{\det S}$ 而非 $\det S$ — 体积 $23/6$ 非负.

2. (§4.10 主题) 计算 $\displaystyle I = \int_\Omega z\,dV$ 其中 $\Omega = \{(x,y,z) : x^2 + y^2 \le 1,\ 0 \le z \le \sqrt{x^2+y^2}\}$ (即半圆锥 from $z = 0$ to $z = r$).

答案

区域绕 $z$ 轴对称 → 用柱坐标 $\Phi(r, \theta, z) = (r\cos\theta, r\sin\theta, z)$, $\abs{\det[D\Phi]} = r$. $\Omega$ 在 $(r,\theta,z)$ 空间: $$\widetilde{\Omega} = \{(r,\theta, z) : 0 \le r \le 1,\ 0 \le \theta \le 2\pi,\ 0 \le z \le r\}.$$

积分: $$I = \int_0^{2\pi}\!\int_0^1\!\int_0^r z\cdot r\,dz\,dr\,d\theta = 2\pi \int_0^1 r\cdot \frac{r^2}{2}\,dr = \pi\int_0^1 r^3\,dr = \pi\cdot\frac{1}{4} = \boxed{\pi/4}.$$

陷阱检查: (a) $\abs{\det[D\Phi]} = r$ 是柱坐标 Jacobian (不是 $r^2$ 球坐标). (b) $\theta$ 上下界 $[0, 2\pi]$ 不是 $[0, 2\pi)$ — 在积分意义下 boundary 测度零 = 无所谓; 写哪个都对.

3. (§4.11 主题) 设 $f(x) = (-\ln x)\cdot \mathbf{1}_{(0,1]}(x)$. 证 $f \in \IL(\R) \setminus \IR(\R)$, 并求 $\int f\, dV_L$.

答案

$f \notin \IR$: $f$ 在 $0$ 邻域无界 ($-\ln x \to +\infty$ as $x \to 0^+$). Topic 02 的 Riemann 框架只接受有界函数, 故 $f \notin \IR(\R)$.

$f \in \IL$ via brick decomposition: 取 dyadic 区间 $I_k = (1/2^k, 1/2^{k-1}]$ for $k \ge 1$. $\bigcup_k I_k = (0, 1]$, 不交. 定义 $$f_k(x) := (-\ln x)\cdot \mathbf{1}_{I_k}(x).$$ 每个 $f_k$ 有界 (在 $I_k$ 上 $-\ln x \in [\ln 2 \cdot (k-1), \ln 2 \cdot k]$) 且 $\supp f_k$ 有界 ($\subset (0, 1]$), $\Disc(f_k) = $ 端点测度零, 故 $f_k \in \IR$. $\sum_k f_k(x) = f(x)$ 对每个 $x$ (Daniell-Stone 等价定义自动给).

验证 $\sum_k \int |f_k|\, dV < \infty$. 由 $-\ln x > 0$ on $(0, 1)$, $\abs{f_k} = f_k$. $$\begin{aligned} \int f_k\, dV &= \int_{I_k} (-\ln x)\, dx = [-x\ln x + x]_{1/2^k}^{1/2^{k-1}} \\ &= \left(\frac{1}{2^{k-1}} - \frac{1}{2^{k-1}}\ln\frac{1}{2^{k-1}}\right) - \left(\frac{1}{2^k} - \frac{1}{2^k}\ln\frac{1}{2^k}\right) \\ &= \frac{1}{2^{k-1}} - \frac{1}{2^k} + (k-1)\ln 2 \cdot \frac{1}{2^{k-1}} - k\ln 2 \cdot \frac{1}{2^k}. \end{aligned}$$ 第一段 (无 $\ln 2$): $\sum_k (1/2^{k-1} - 1/2^k) = \sum_k 1/2^k = 1$ (telescoping). 第二段 ($\ln 2$ 系数): $\ln 2 \sum_k [(k-1)/2^{k-1} - k/2^k]$, 也 telescoping, 总和 = 0.

故 $\sum_k \int f_k = 1 + 0 = 1 < \infty$. 由 Lebesgue 定义, $f \in \IL$ 且 $$\int f\, dV_L = \sum_k \int f_k\, dV = \boxed{1}.$$ (匹配 improper Riemann 计算: $\int_0^1 (-\ln x)\, dx = [x(1 - \ln x)]_0^1 = 1$.)

4. (§4.9 + §4.10 跨章节) 计算 $\displaystyle I = \int_E e^{-(x^2 + xy + y^2)}\, dV$ 其中 $E$ 是整个 $\R^2$.

答案

策略: 二次型对角化 (linear CoV, §4.9) → 极坐标 (nonlinear CoV, §4.10).

Step 1 (Linear CoV 对角化). 二次型矩阵 $M = \begin{pmatrix} 1 & 1/2 \\ 1/2 & 1 \end{pmatrix}$, $\det M = 1 - 1/4 = 3/4$, 特征值 $\lambda_1 = 3/2, \lambda_2 = 1/2$ (从 $\lambda^2 - 2\lambda + 3/4 = 0$). 取 $T = S\mathrm{diag}(\sqrt{2/3}, \sqrt{2})$ 其中 $S \in O(2)$ 对角化 $M$. 则 $T^T M T = I_2$, 即 $Q(T(u,v)) = u^2 + v^2$. $\abs{\det T} = 1\cdot \sqrt{(2/3)\cdot 2} = \sqrt{4/3} = 2/\sqrt{3}$.

由 Thm A.2 (Linear CoV): $$I = \abs{\det T}\int_{\R^2} e^{-(u^2+v^2)}\, dV = \frac{2}{\sqrt{3}}\int_{\R^2} e^{-(u^2+v^2)}\, dV.$$

Step 2 (极坐标算右边). $u = r\cos\theta, v = r\sin\theta$, $\abs{\det[DP]} = r$: $$\int_{\R^2} e^{-(u^2+v^2)}\, dV = \int_0^{2\pi}\!\int_0^\infty e^{-r^2}\cdot r\,dr\,d\theta = 2\pi\cdot\frac{1}{2} = \pi.$$ (内层 $\int_0^\infty r e^{-r^2}\,dr = [-e^{-r^2}/2]_0^\infty = 1/2$.)

合起来: $$I = \frac{2}{\sqrt{3}}\cdot\pi = \boxed{\frac{2\pi}{\sqrt{3}}}.$$

Sanity check via 公式: 正定二次型 $\int_{\R^n} e^{-x^T M x}\, dV = \pi^{n/2}/\sqrt{\det M}$. 此处 $n = 2$, $\det M = 3/4$, 给 $\pi/\sqrt{3/4} = \pi\cdot 2/\sqrt{3} = 2\pi/\sqrt{3}$ ✓.

5. (§4.10 + §4.11 跨章节) 设 $f(x, y) = 1/\sqrt{x^2 + y^2}$ on 单位圆盘 $D = \{(x,y) : x^2+y^2 \le 1\}$, 0 elsewhere. (a) 证 $f \notin \IR(\R^2)$ (因无界). (b) 证 $f \in \IL(\R^2)$ via brick decomposition 并求 $\int f\,dV_L$.

答案

(a) $f \notin \IR$: $f$ 在 $(0,0)$ 邻域无界, Topic 02 框架要求 $f$ 有界, 故 $f \notin \IR(\R^2)$.

(b) $f \in \IL$ via dyadic annular brick decomposition. 取 annular 区域 $A_k = \{(x,y) : 1/2^k < \sqrt{x^2+y^2} \le 1/2^{k-1}\}$ for $k \ge 1$. $\bigcup_k A_k = D \setminus \{(0,0)\}$, 不交.

定义 $f_k(x,y) := \frac{1}{\sqrt{x^2+y^2}}\cdot \mathbf{1}_{A_k}(x,y)$. 在 $A_k$ 上 $f$ 有界 ($2^{k-1} \le f \le 2^k$), $\supp f_k$ 紧致, Disc 在边界测度零, 故 $f_k \in \IR(\R^2)$. $\sum_k f_k = f$ 处处 (除 $(0,0)$ 单点测度零).

计算 $\int f_k\, dV$ (极坐标, 由 §4.10): $$\int f_k\, dV = \int_0^{2\pi}\!\int_{1/2^k}^{1/2^{k-1}} \frac{1}{r}\cdot r\, dr\, d\theta = 2\pi\int_{1/2^k}^{1/2^{k-1}} dr = 2\pi\cdot\left(\frac{1}{2^{k-1}} - \frac{1}{2^k}\right) = 2\pi\cdot\frac{1}{2^k}.$$ 注意 polar Jacobian $r$ 跟 $1/r$ 抵消 — 这正是 $1/\sqrt{x^2+y^2}$ 在 $\R^2$ 比在 $\R$ 上"更可积"的关键. 求和: $$\sum_k \int |f_k|\, dV = \sum_k \int f_k\, dV = \sum_{k=1}^\infty \frac{2\pi}{2^k} = 2\pi\cdot 1 = 2\pi < \infty.$$ 由 Lebesgue 定义, $f \in \IL$ 且 $\int f\, dV_L = \boxed{2\pi}$.

对比 §4.11 反例: 1D 类比 $1/\sqrt{|x|}$ on $[-1,1]$ — $\int_L = 4$ (由 §4.11 reference brick). 但 1D 平方 $1/|x|$ 不在 $\IL$ — 因为去掉 $\sqrt$ 之后 $\int r^{-2}\cdot r\,dr$ 在 0 附近发散 (1D). 此处 2D 中 $f = 1/\sqrt{x^2+y^2}$, Jacobian $r$ 把 $1/r$ "救活". 这是 polar 几何与可积性的精妙交互.
