§3 Fourier & Laplace 速查 (计算题主战场之一)

§3.1 Fourier 变换: 约定与公式

⚠ 本课程符号约定 (与多数教材相反!)

$$\FT{f}(\zeta) = \int_{\R^n} f(x)\,e^{+2\pi i\,\zeta\cdot x}\,dx, \qquad f(x) = \int_{\R^n} \FT{f}(\zeta)\,e^{-2\pi i\,\zeta\cdot x}\,d\zeta.$$

正变换是 $+2\pi i$, 逆变换是 $-2\pi i$ (H&H / 本课 Lec 10.3 板书约定), 与 Stein、Folland 等正好反号; 无 $1/(2\pi)$ 因子。从课外资料抄公式前必须核对约定, 否则微分公式差正负号。

  • 微分 → 乘法: $\widehat{\partial_{x_j} f}(\zeta) = -2\pi i\,\zeta_j\,\FT{f}(\zeta)$; 迭代得 $\widehat{\Delta f}(\zeta) = -4\pi^2|\zeta|^2\,\FT{f}(\zeta)$。
  • 卷积定理: $(f*g)(x) = \int f(x-y)\,g(y)\,dy$, 则 $\widehat{f*g} = \FT{f}\cdot\FT{g}$。配合 FT 在 $\IL(\R^n)$ 上单射: $\FT{h} = \FT{f}\,\FT{g} \Rightarrow h = f*g$。
  • ★★ Gaussian: $\widehat{e^{-ax^2}}(\zeta) = \sqrt{\pi/a}\;e^{-\pi^2\zeta^2/a}$; $n$ 维 $\widehat{e^{-a|x|^2}}(\zeta) = (\pi/a)^{n/2} e^{-\pi^2|\zeta|^2/a}$。
  • ★ Double-exp: $\widehat{e^{-a|x|}}(\zeta) = \dfrac{2a}{a^2 + 4\pi^2\zeta^2}$; 特别地 $a=1$: $\dfrac{1}{1+4\pi^2\zeta^2} = \tfrac12\,\widehat{e^{-|x|}}(\zeta)$。
  • Indicator: $\widehat{\mathbf{1}_{[-A,A]}}(\zeta) = \dfrac{\sin(2\pi\zeta A)}{\pi\zeta}$ ($\zeta=0$ 时 $=2A$)。

§3.2 模板 A: 热方程 (Lec 10.3 Example I)

热方程四步流程 — $u_t = \Delta u$, $u(0,x) = f(x)$
  1. 对 $x$ 取 FT: $\partial_t \FT{u}(\zeta,t) = -4\pi^2|\zeta|^2\,\FT{u}(\zeta,t)$ — PDE 变成 (每个 $\zeta$ 一条) ODE。
  2. 解 ODE: $\FT{u}(\zeta,t) = C(\zeta)\,e^{-4\pi^2|\zeta|^2 t}$; 初值给 $C(\zeta) = \FT{f}(\zeta)$。
  3. 识别 frequency 因子是谁的 FT: 由 Gaussian 公式 (取 $a = \tfrac{1}{4t}$), $e^{-4\pi^2|\zeta|^2 t} = \FT{K_t}(\zeta)$, 其中热核 $K_t(x) = (4\pi t)^{-n/2}\,e^{-|x|^2/(4t)}$。
  4. 卷积定理 + 单射性收尾: $$u(x,t) = (K_t * f)(x) \;\overset{n=1}{=}\; \int_\R \frac{e^{-(x-y)^2/(4t)}}{\sqrt{4\pi t}}\,f(y)\,dy.$$

§3.3 模板 B: Poisson 型方程 (Lec 10.3 Example II)

$-u'' + u = f$ on $\R$

FT 两边: $(-2\pi i\zeta)^2$ 给 $-\FT{u''} = +4\pi^2\zeta^2\FT{u}$, 故 $$(1 + 4\pi^2\zeta^2)\,\FT{u}(\zeta) = \FT{f}(\zeta) \;\Longrightarrow\; \FT{u}(\zeta) = \frac{\FT{f}(\zeta)}{1+4\pi^2\zeta^2}.$$ 识别 $\dfrac{1}{1+4\pi^2\zeta^2} = \tfrac12\widehat{e^{-|x|}}(\zeta)$, 由卷积定理 + 单射性: $$u(x) = \tfrac12\,(e^{-|\cdot|} * f)(x) = \tfrac12\int_\R e^{-|x-y|}\,f(y)\,dy.$$ 同型变体 (Yukawa, $\R^3$): $-\Delta u + m^2 u = f$ → $G(x) = e^{-m|x|}/(4\pi|x|)$, 见 Topic 17。

§3.4 Laplace 变换: 公式表

  • 定义: $\LT(f)(s) = \displaystyle\int_0^\infty f(t)\,e^{-st}\,dt$ (在 $s$ 足够大时收敛)。
  • ★★ 求导公式 (ODE 的引擎): $\LT(f')(s) = s\LT(f)(s) - f(0)$; $\;\LT(f'')(s) = s^2\LT(f)(s) - s f(0) - f'(0)$。初值项绝对不能漏。
  • 指数: $\LT(e^{-at})(s) = \dfrac{1}{s+a}$ ($s > -a$); 单项式×指数: $\LT(t^m e^{-at})(s) = \dfrac{m!}{(s+a)^{m+1}}$; 特例 $\LT(1) = 1/s$, $\LT(t^m) = m!/s^{m+1}$。
  • 三角: $\LT(\sin bt)(s) = \dfrac{b}{s^2+b^2}$, $\quad\LT(\cos bt)(s) = \dfrac{s}{s^2+b^2}$。
  • ★ $s$-求导技巧 (Lec 10.3 aside): $\LT(t f(t))(s) = -\dfrac{d}{ds}\LT(f)(s)$。推论: $\LT(t\sin t) = \dfrac{2s}{(s^2+1)^2}$, $\;\LT(t\cos t) = \dfrac{s^2-1}{(s^2+1)^2}$, $\;\LT\!\big(\tfrac12(\sin t - t\cos t)\big) = \dfrac{1}{(s^2+1)^2}$ — 逆变换 $\dfrac{1}{(s^2+1)^2}$ 全靠它。
  • 卷积 (Laplace 版): $(f*g)(t) = \displaystyle\int_0^t f(t-y)g(y)\,dy$, $\LT(f*g) = \LT(f)\LT(g)$。注意积分限是 $[0,t]$, 与 FT 版不同。

§3.5 模板 C: Laplace 解 ODE 四步 (Lec 10.3 两例)

流程: 变换 → 代数 → 部分分式 → 查表逆变换

例 1 (一阶): $y' + 3y = e^{-t}$, $y(0) = 2$。 $$\big(s\LT(y) - 2\big) + 3\LT(y) = \frac{1}{s+1} \;\Rightarrow\; \LT(y) = \frac{1}{(s+1)(s+3)} + \frac{2}{s+3} = \frac{1/2}{s+1} + \frac{3/2}{s+3},$$ 故 $y(t) = \tfrac12 e^{-t} + \tfrac32 e^{-3t}$ (验 $y(0) = 2$ ✓)。

例 2 (二阶, 共振): $y'' + y = \sin t$, $y(0)=0$, $y'(0)=1$。 $$\big(s^2\LT(y) - 1\big) + \LT(y) = \frac{1}{s^2+1} \;\Rightarrow\; \LT(y) = \frac{1}{s^2+1} + \frac{1}{(s^2+1)^2}.$$ 第二项用 $s$-求导技巧逆变换: $\dfrac{1}{(s^2+1)^2} = \LT\big(\tfrac12(\sin t - t\cos t)\big)$, 故 $$y(t) = \sin t + \tfrac12(\sin t - t\cos t) = \tfrac32\sin t - \tfrac12\,t\cos t.$$ (验: $y(0)=0$ ✓, $y'(t) = \cos t + \tfrac{t}{2}\sin t$, $y'(0)=1$ ✓。$t\cos t$ 的线性增长 = 共振现象。)

收尾必做: 把初值代回答案检查 — 10 秒钟, 抓住 90% 的部分分式手滑。

§4 曲线速查 (κ 与 τ)

🆕 OH 情报对本节的影响

Frenet frame 的理论装置 (Gram–Schmidt 构造、distinguished frame 存在唯一性、$\omega$ 矩阵推导) 大概率不考, 可降低优先级; 但曲率计算是官方点名的计算板块之一 — 下表的 $\kappa, \tau$ 公式、helix 标志值、以及 "τ 分母是叉积平方" 这条易错点仍是必背核心。$\T, \NN, \B$ 的定义留着即可 (空间曲线公式的语言), Frenet 方程组扫一眼有印象就行。

  • ★★ 平面曲线 (signed): $c(t) = (x(t), y(t))$, $$\kappa = \frac{x'y'' - y'x''}{\big((x')^2+(y')^2\big)^{3/2}} = \frac{\det(\dot c, \ddot c)}{|\dot c|^3}.$$ 符号: $+$ ⟺ 向左转 (向 $e_2 = R_{90°}e_1$ 方向弯)。
  • 函数图象: $c(t) = (t, f(t))$ → $\kappa = \dfrac{f''}{(1+(f')^2)^{3/2}}$; 半径 $r$ 的圆: $\kappa \equiv 1/r$。
  • ★★ 空间曲线 (unsigned $\kappa$, signed $\tau$): $$\kappa = \frac{|\dot c \times \ddot c|}{|\dot c|^3} \;(>0), \qquad \tau = \frac{\det(\dot c, \ddot c, \dddot c)}{|\dot c \times \ddot c|^2}.$$ ⚠ $\tau$ 的分母是 $|\dot c\times\ddot c|^2$, 不是 $|\dot c|^6$ — 最高频考场错误。
  • Frenet 三标架: $\T = \dot c/|\dot c|$, $\;\NN = \dot\T/|\dot\T|$, $\;\B = \T\times\NN$ (右手系)。Frenet 方程 (unit-speed): $\T' = \kappa\NN$, $\;\NN' = -\kappa\T + \tau\B$, $\;\B' = -\tau\NN$。
  • ★ Helix 标志值: $c(t) = (a\cos t, a\sin t, bt)$ → $\kappa = \dfrac{a}{a^2+b^2}$, $\tau = \dfrac{b}{a^2+b^2}$, 双双常数 (且是 $\kappa,\tau$ 全常数的唯一空间曲线, up to rigid motion)。
  • 参数化不变性: $\kappa, \tau$ 只依赖 image + orientation — 公式已含 $|\dot c|$ 修正因子, 不必先化 unit-speed; 选最好算的参数化即可。
  • 退化警告: 直线 $\ddot c = 0$ → Frenet frame 不存在, $\tau$ 是 $0/0$ 无定义; 不要写 "直线 $\tau = 0$"。

详细推导与练习: Topic 19 (Frenet frame 构造) / Topic 20 (曲率公式 + 证明)。

§5 曲面速查 (Final Review 课主角)

§5.1 计算流水线 (背熟, 每道曲面题都走这条线)

曲面六步流水线 — $X: U \subset \R^2 \to \R^3$
  1. $X_u, X_v$ → 第一基本形式 $\I = \begin{pmatrix} E & F \\ F & G\end{pmatrix}$, $\;E = X_u\!\cdot\!X_u$, $F = X_u\!\cdot\!X_v$, $G = X_v\!\cdot\!X_v$。
  2. 法向 $\nu = \dfrac{X_u\times X_v}{|X_u\times X_v|}$, 其中 $|X_u\times X_v| = \sqrt{EG - F^2}$ (Lagrange)。面积 $= \iint_\Omega \sqrt{EG-F^2}\,du\,dv$。
  3. $X_{uu}, X_{uv}, X_{vv}$ → 第二基本形式 $\II = \begin{pmatrix} L & M \\ M & N\end{pmatrix}$, $\;L = X_{uu}\!\cdot\!\nu$, $M = X_{uv}\!\cdot\!\nu$, $N = X_{vv}\!\cdot\!\nu$。
  4. Gauss 曲率 $K = \dfrac{LN - M^2}{EG - F^2}$; 平均曲率 $H = \dfrac{EN - 2FM + GL}{2(EG - F^2)}$。
  5. Shape operator $S = \I^{-1}\II$; 主曲率 $k_1, k_2$ = $S$ 的特征值 (恒为实); $K = k_1 k_2 = \det S$, $H = \tfrac{k_1+k_2}{2} = \tfrac12\operatorname{tr} S$。
  6. 若 $\I, \II$ 同时对角 ($F = M = 0$): 直接 $k_1 = L/E$, $k_2 = N/G$ — 球面 / 旋转面 / torus 都走这条捷径。
Theorema Egregium (概念题考点)

$K$ 内蕴 — 只依赖 $\I$, 局部等距不变 ("测量者不出曲面就能算出 $K$"); $H$ 不内蕴。经典见证: 圆柱与平面局部等距, 都有 $K = 0$, 但圆柱 $H \ne 0$ 平面 $H = 0$。另注意 $K$ 不依赖 $\nu$ 方向 ($\II$ 的两行同时变号, $LN - M^2$ 不变), $H$ 随 $\nu$ 翻转变号 — 报 $H$ 时要么说明法向, 要么报 $|H|$。

§5.2 Graph surface $z = f(x,y)$ 特化公式 (Final Review "important example")

$X(x,y) = (x, y, f(x,y))$, 记 $W = \sqrt{1 + f_x^2 + f_y^2}$

$$\I = \begin{pmatrix} 1+f_x^2 & f_x f_y \\ f_x f_y & 1 + f_y^2 \end{pmatrix}, \qquad \nu = \frac{(-f_x,\, -f_y,\, 1)}{W}, \qquad \II = \frac{1}{W}\begin{pmatrix} f_{xx} & f_{xy} \\ f_{xy} & f_{yy} \end{pmatrix},$$ $$K = \frac{f_{xx}f_{yy} - f_{xy}^2}{W^4} = \frac{f_{xx}f_{yy} - f_{xy}^2}{(1+f_x^2+f_y^2)^2}, \qquad H = \frac{(1+f_y^2)f_{xx} - 2f_xf_yf_{xy} + (1+f_x^2)f_{yy}}{2\,W^3}.$$

口诀: $K$ 分子是 "Hessian 行列式", 分母 $W^4$; $H$ 分子交叉配权, 分母 $2W^3$。(勘误: Final Review 板书的 $H$ 漏写了分母的 2; 用 $H = \frac{EN-2FM+GL}{2(EG-F^2)}$ 代入可验, minimal set 结论不受影响。)

Worked: 双曲抛物面 $z = \frac{x^2 - y^2}{2}$ (Final Review 例题)

$f_x = x$, $f_y = -y$, $f_{xx} = 1$, $f_{yy} = -1$, $f_{xy} = 0$, $W^2 = 1+x^2+y^2$: $$K = \frac{(1)(-1) - 0}{(1+x^2+y^2)^2} = \frac{-1}{(1+x^2+y^2)^2} < 0 \;\text{处处 (鞍面)}, \qquad H = \frac{(1+y^2) - (1+x^2)}{2(1+x^2+y^2)^{3/2}} = \frac{y^2 - x^2}{2(1+x^2+y^2)^{3/2}}.$$ Minimal points ($H = 0$): $y = \pm x$ — 即曲面上的两条直线 $\{(x, \pm x, 0)\}$。定义: $H(p) = 0$ 的点叫 minimal point; 处处 minimal 的曲面叫 minimal surface。同法可练椭圆抛物面 $z = \frac{x^2+y^2}{2}$ (无 minimal point: $H$ 分子 $= 2 + x^2 + y^2 > 0$)。

§5.3 Torus 全套结果 (Final Review 压轴例题 — 必会)

$X(u,v) = \big((R + r\cos v)\cos u,\; (R + r\cos v)\sin u,\; r\sin v\big)$, $\;R > r > 0$

$u$ = 大圆角, $v$ = 小圆角 ($v = 0$ 最外圈, $v = \pi$ 最内圈)。计算走流水线 (取外法向 $\nu = (\cos v\cos u, \cos v\sin u, \sin v)$):

$$\I = \begin{pmatrix}(R+r\cos v)^2 & 0\\ 0 & r^2\end{pmatrix}, \qquad \II = \begin{pmatrix}-(R+r\cos v)\cos v & 0\\ 0 & -r\end{pmatrix},$$ $$k_1 = \frac{-\cos v}{R + r\cos v}, \quad k_2 = -\frac1r, \qquad \boxed{\,K = k_1 k_2 = \frac{\cos v}{r\,(R + r\cos v)}\,}, \qquad H = -\frac{R + 2r\cos v}{2r(R + r\cos v)}.$$

$K$ 的符号地理: 外半圈 ($|v| < \pi/2$) $K > 0$ (椭圆型); 内半圈 $K < 0$ (鞍型); 顶/底圈 ($v = \pm\pi/2$) $K = 0$。(板书取内法向, $\II$ 与 $H$ 整体反号, $K$ 与 minimal 条件不变。)

Minimal points 三分讨论 ($H = 0 \iff \cos v = -\frac{R}{2r}$):

  • $R > 2r$: $-\frac{R}{2r} < -1$ — 无 minimal point ("管子太细/洞太大");
  • $R = 2r$: $\cos v = -1$, $v = \pi$ — minimal points = 最内圈 $\{((R-r)\cos u, (R-r)\sin u, 0)\}$;
  • $r < R < 2r$: $v = \pm\arccos(-\frac{R}{2r})$ — 两个圆周。

§5.4 标志曲面结果表 (报数即可的"常识")

  • 球面 (半径 $R$): $K = 1/R^2$, $|H| = 1/R$, 处处脐点 ($k_1 = k_2$)。
  • 圆柱面 (半径 $r$): $K = 0$, $|H| = \frac{1}{2r}$ — "内蕴平、外在弯", 与平面局部等距。
  • 鞍面 $z = uv$ 在原点: $K = -1$, $H = 0$, $k_{1,2} = \pm 1$ — minimal 且双曲。
  • 平面: $\II = 0$, $K = H = 0$。
  • ⚠ $K = 0$ 不等于平面 — 圆柱、圆锥都 $K=0$ (developable); $K = 0$ 只说"内蕴 flat"。

详细推导与练习: Topic 21 ($\I$/弧长/面积) / Topic 22 ($\II$/$K$/$H$/shape operator)。

§6 收敛定理速查 (~30% 分值, 全是模板题)

🆕 OH 情报: Fatou 是本板块的钦点主角

教授在 OH 明示 "be prepared to use Fatou's theorem — it's actually the most useful"。落实到行动: ① Fatou 的完整证明骨架背到默写 — 令 $g_m := \inf_{k \ge m} f_k$, 则 $g_m$ 单调增、$g_m \le f_m$, 对 $(g_m)$ 套 MCT, 再由 $\int g_m \le \int f_m$ 取 $\liminf$; ② 两个变体 (a.e. 版 P2、与 generalized DCT 的双侧 Fatou 用法 P8) 都过一遍; ③ 记牢不等号方向与非负前提 (§7 易错点 8)。

§6.1 两大主定理 + 四个推论

MCT for Lebesgue (H&H Thm 4.11.18)

递增版: $(f_k) \subset \IL(\R^n)$, (H1) $0 \le f_1 \le_L f_2 \le_L \cdots$ (a.e. 逐对), (H2) $\sup_k \int f_k\,dV_L < \infty$ ⟹ $f := \lim f_k \in \IL$ 且 $\int f\,dV_L = \lim_k \int f_k\,dV_L$。

递减版 (Practice MT2 P3): $f_k \downarrow f$ a.e., 某个 $\int f_k$ 有限 ⟹ 同结论。证: $h_k := f_1 - f_k$ 翻成递增套上版。

DCT for Lebesgue (H&H Thm 4.11.19)

$(f_k) \subset \IL(\R^n)$, (H1) $f_k \to f$ a.e., (H2) 存在 dominator $F \in \IL$ 使 $|f_k| \le_L F$ 对所有 $k$ ⟹ $f \in \IL$, 且 $\int|f_k - f|\,dV_L \to 0$, 特别地 $\int f_k \to \int f$。

⚠ 与 DCT for Riemann (Thm 4.11.6) 的区别: Riemann 版要求 $f \in \IR$ 是付费假设 (要自己验); Lebesgue 版白送 $f \in \IL$。

  • Fatou 引理 (≡ Practice MT2 P1): $f_k \ge 0$ in $\IL$ (且 $\liminf\int f_k < \infty$) ⟹ $\int \liminf f_k \le \liminf \int f_k$。证: $g_m := \inf_{k \ge m} f_k$ 单调增, 套 MCT。记不等号方向: 积分会"漏质量", 不会"凭空多"。
  • Fatou for a.e. limit (≡ P2): $f_k \ge 0$, $f_k \to f$ a.e. ⟹ $\int f \le \liminf \int f_k$ (因 a.e. 收敛 ⟹ $\liminf f_k = f$ a.e.)。
  • Generalized DCT (≡ P8): dominator 可变 — $|f_k| \le g_k$, $g_k \to g$ a.e., $\int g_k \to \int g < \infty$, $f_k \to f$ a.e. ⟹ $\int f_k \to \int f$。证: Fatou 同时用于 $g_k + f_k \ge 0$ 和 $g_k - f_k \ge 0$, 两边夹。
  • Scheffé (≡ P9): $f_k \to f$ a.e. 时, $\int|f_k| \to \int|f| \iff \int|f_k - f| \to 0$。实质方向证: $h_k := |f_k - f|$ 被 $g_k := |f_k| + |f|$ 控制, 套 Generalized DCT。

§6.2 选定理决策树 + 答题模板

看到 "$\lim \int f_k \overset{?}{=} \int \lim f_k$" 类题, 按序自问:
  1. 序列单调吗? (a.e. 递增/递减) → MCT。递增验 $\sup\int f_k < \infty$; 递减验某 $\int f_k$ 有限。
  2. 能找到固定可积 dominator $F$ 吗? ($|f_k| \le F$, $\int F < \infty$) → DCT。$F$ 常见来源: 题目给的界 × 指示函数、首项 $|f_1|$、$\min/\max$ 组合。
  3. 只有非负性、问的是不等式? → Fatou ($\le \liminf$ 方向)。
  4. dominator 自己也在动? ($|f_k| \le g_k$, $g_k$ 收敛且积分收敛) → Generalized DCT。
  5. 问 $\int|f_k - f| \to 0$ ($L^1$ 收敛)? → Scheffé (验 $\int|f_k| \to \int|f|$)。

答题模板 (4–10 行): ① 点名定理; ② 逐条验 hypotheses — 单调性 / dominator 的可积性 ($\int F < \infty$ 要算出来或给界) / a.e. 收敛 (求 pointwise limit); ③ 引用结论收尾。教授 MT2 原话: "if you apply this, you immediately get the answer" — final 的收敛定理题同样是识别 + 验设 + 套用。

两个必背反例 (考"为什么需要 hypotheses"时直接搬)
  • Traveling box $f_k = \mathbf{1}_{[k, k+1]}$: 逐点 $\to 0$ 但 $\int f_k \equiv 1 \ne 0$。质量逃向 $\infty$; 任何 dominator 必须 $\ge \mathbf{1}_{[1,\infty)}$ 不可积 → DCT 的 (H2) 不可修复。
  • Tall thin spike $f_k = k\,\mathbf{1}_{(0, 1/k]}$: 逐点 $\to 0$ 但 $\int f_k \equiv 1$。质量集中在 $0$; dominator 必须 $\ge 1/x$ 不可积。
  • 两例中 Fatou 不等式 $0 = \int\lim \le \liminf\int = 1$ 严格 — Fatou 的 "$\le$" 不能升级成 "$=$" 的活见证。

完整理论链 + 更多例题: MT2-C (Lebesgue 积分); 五道原题详解: Practice MT2 P1/P2/P3/P8/P9。

§7 易错点 Top 10 (考前一晚扫一遍)

  • 1. FT 符号约定: 本课正变换 $e^{+2\pi i \zeta x}$、逆变换 $e^{-2\pi i \zeta x}$ — 与 Stein/Folland 反号; 微分公式因此是 $\widehat{\partial f} = -2\pi i\zeta\FT{f}$。抄外部资料先核对。
  • 2. Laplace 漏初值项: $\LT(f') = s\LT(f) - f(0)$, $\LT(f'') = s^2\LT(f) - sf(0) - f'(0)$ — 初值不是零时漏掉直接全错。
  • 3. $\tau$ 的分母: $\tau = \det(\dot c,\ddot c,\dddot c)\,/\,|\dot c\times\ddot c|^2$, 不是 $|\dot c|^6$。
  • 4. 面积元: $\sqrt{EG - F^2}$, 不是 $\sqrt{EG}$ — $F \ne 0$ 的参数化 (如 graph) 必须减 $F^2$。
  • 5. $H$ 的分母有 2: $H = \frac{EN - 2FM + GL}{2(EG-F^2)} = \frac{k_1+k_2}{2}$ 是平均; 且 $H$ 随 $\nu$ 翻转变号 ($K$ 不会)。
  • 6. 平面 $\kappa$ signed, 空间 $\kappa > 0$ (符号转移给 $\tau$); 直线 Frenet 失效, $\tau$ 无定义。
  • 7. DCT 的 dominator 必须可积: 写 "$|f_k| \le F$" 不够, 必须验 $\int F < \infty$ — traveling box / spike 就死在这一步。
  • 8. Fatou 方向: $\int\liminf \le \liminf\int$ (质量只会漏不会多), 且需 $f_k \ge 0$; 反向不等式不成立。
  • 9. Gaussian FT 的 $\pi$ 位置: $\widehat{e^{-ax^2}} = \sqrt{\pi/a}\,e^{-\pi^2\zeta^2/a}$ — 指数里是 $\pi^2/a$, 系数是 $\sqrt{\pi/a}$; 热核识别 $a = \frac{1}{4t}$ 时别代错。
  • 10. $\kappa, \tau$ 公式不需要 unit-speed: 公式已含 $|\dot c|$ 因子; helix $(a\cos t, a\sin t, bt)$ 速度是 $\sqrt{a^2+b^2}$ 不是 1, 但直接套公式即可, 不要先费力重参数化。
