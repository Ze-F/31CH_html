MATH 31CH Final — Side 1: Fourier · Laplace · 解方程 · 收敛定理 · 径向积分

FT 约定与运算规则 (正变换 +2πi !)

$\FT{f}(\xi) = \int_{\R^n} f(x)\,e^{+2\pi i \xi\cdot x} dx$, $\;f(x) = \int \FT{f}(\xi)\,e^{-2\pi i \xi\cdot x} d\xi$ (与 Stein/Folland 反号, 无 $1/2\pi$)

  • 微分→乘法: $\widehat{\partial_{x_j} f} = -2\pi i \xi_j \FT{f}$; $\;\widehat{\Delta f} = -4\pi^2|\xi|^2 \FT{f}$
  • 乘法→微分: $\partial_{\xi_j}\FT{f} = 2\pi i\,\widehat{x_j f}$
  • 平移: $\widehat{f(\cdot - a)}(\xi) = e^{2\pi i a\xi}\FT{f}(\xi)$; 调制: $\widehat{e^{2\pi i b x}f}(\xi) = \FT{f}(\xi + b)$
  • 缩放: $\widehat{f(ax)}(\xi) = \frac{1}{|a|}\FT{f}(\xi/a)$
  • 卷积: $(f{*}g)(x) = \int f(x{-}y)g(y)dy$, $\widehat{f{*}g} = \FT{f}\FT{g}$; FT 在 $\IL$ 上单射 (逆推 $u = K * f$ 的依据)

FT 值表 (PF Part I 全部出过)

$f(x)$$\FT{f}(\xi)$
$\mathbf 1_{[-a,a]}$$\dfrac{\sin(2\pi a\xi)}{\pi\xi}$ ($\xi{=}0$: $2a$)
$\mathbf 1_{[-a,a]\times[-b,b]}$张量积: $\frac{\sin(2\pi a\xi_1)}{\pi\xi_1}\cdot\frac{\sin(2\pi b\xi_2)}{\pi\xi_2}$
$e^{-a|x|}$$\dfrac{2a}{a^2+4\pi^2\xi^2}$; 注意 $\frac{1}{1+4\pi^2\xi^2} = \frac12\widehat{e^{-|x|}}$
$e^{-ax^2}$$\sqrt{\pi/a}\;e^{-\pi^2\xi^2/a}$
$e^{-\pi x^2}$$e^{-\pi\xi^2}$ (自对偶!)
$e^{-a|x|^2}$ ($n$D)$(\pi/a)^{n/2}e^{-\pi^2|\xi|^2/a}$

Gauss 积分: $\int_\R e^{-\pi x^2}dx = 1$; $\int_\R e^{-x^2}dx = \sqrt\pi$; $\int_\R e^{-ax^2}dx = \sqrt{\pi/a}$

FT 解 PDE 模板 (PF Part III.1/III.2)

通用算法: FT 两边 → 常系数算子变乘法 (symbol) → 解出 $\FT{u}$ → 识别核 + 卷积定理 + 单射性逆推。

热方程 $u_t = u_{xx}$, $u(x,0){=}f$: $\partial_t\FT u = -4\pi^2\xi^2\FT u$ → $\FT u = \FT f e^{-4\pi^2\xi^2 t}$ → 识别 $G_t(x) = \frac{e^{-x^2/4t}}{\sqrt{4\pi t}}$, $\FT{G_t} = e^{-4\pi^2 t\xi^2}$ → $u(x,t) = (G_t * f)(x) = \frac{1}{\sqrt{4\pi t}}\int e^{-(x-y)^2/4t} f(y)dy$ ($n$D: $(4\pi t)^{-n/2}$)

Poisson 型 $-u'' + u = f$: $(1+4\pi^2\xi^2)\FT u = \FT f$ → $u = \tfrac12 e^{-|\cdot|} * f = \tfrac12\int e^{-|x-y|}f(y)dy$

$\R^3$ Newtonian: $-\Delta u = f$ → $u = \frac{1}{4\pi}\int\frac{f(y)}{|x-y|}dy$; Yukawa $-\Delta u + m^2 u = f$ → 核 $\frac{e^{-m|x|}}{4\pi|x|}$

Laplace 值表与规则

$\LT(f)(s) = \int_0^\infty f(t)e^{-st}dt$

  • $\LT(f') = s\LT(f) - f(0)$; $\;\LT(f'') = s^2\LT(f) - sf(0) - f'(0)$ 初值勿漏!
  • $\LT(1) = \frac1s$; $\LT(t^m) = \frac{m!}{s^{m+1}}$; $\LT(e^{-at}) = \frac{1}{s+a}$; $\LT(t^m e^{-at}) = \frac{m!}{(s+a)^{m+1}}$
  • $\LT(\sin bt) = \frac{b}{s^2+b^2}$; $\LT(\cos bt) = \frac{s}{s^2+b^2}$; $\LT(\sinh bt) = \frac{b}{s^2-b^2}$; $\LT(\cosh bt) = \frac{s}{s^2-b^2}$
  • 频移: $\LT(e^{-at}f(t))(s) = \LT(f)(s+a)$
  • $s$-求导: $\LT(t f(t)) = -\frac{d}{ds}\LT(f)$ ⟹ $\LT(t\sin t) = \frac{2s}{(s^2+1)^2}$, $\LT(t\cos t) = \frac{s^2-1}{(s^2+1)^2}$, $\frac{1}{(s^2+1)^2} = \LT\big(\tfrac12(\sin t - t\cos t)\big)$
  • 卷积: $(f{*}g)(t) = \int_0^t f(t{-}y)g(y)dy$, $\LT(f{*}g) = \LT(f)\LT(g)$ (积分限 $[0,t]$!)

LT 解 ODE 四步 + PF 两例 (III.3/III.4)

① 取 $\LT$ 带初值 ② 代数解出 $Y(s)$ ③ 部分分式 ④ 查表逆变换; 末尾代回 $y(0), y'(0)$ 验算

  • $y' + 3y = e^{-t}$, $y(0){=}2$: $Y = \frac{2s+3}{(s+1)(s+3)} = \frac{1/2}{s+1} + \frac{3/2}{s+3}$ → $y = \frac12 e^{-t} + \frac32 e^{-3t}$
  • $y'' + y = \sin t$, $y(0){=}0, y'(0){=}1$ (共振): $Y = \frac{1}{s^2+1} + \frac{1}{(s^2+1)^2}$ → $y = \frac32\sin t - \frac t2\cos t$

部分分式速算: $\frac{1}{(s+1)(s+3)} = \frac{1/2}{s+1} - \frac{1/2}{s+3}$ (盖住法: 在极点处求值)

收敛定理全家 (black box 引用)

  • MCT↑ (4.11.18): $0 \le f_k \uparrow$ a.e., $\sup_k\int f_k < \infty$ ⟹ $f := \lim f_k \in \IL$, $\int f = \lim\int f_k$。MCT↓: $f_k \downarrow f$ a.e. + 某 $\int f_k$ 有限 (证: $f_1 - f_k$ 翻转)
  • DCT (4.11.19): $f_k \to f$ a.e. + $|f_k| \le_L F \in \IL$ ⟹ $f \in \IL$, $\int|f_k{-}f| \to 0$, $\int f_k \to \int f$。必须验 $\int F < \infty$
  • Fatou (★ 钦点): $f_k \ge 0$, $\liminf\int f_k < \infty$ ⟹ $\liminf f_k \in \IL$ 且 $\int\liminf f_k \le \liminf\int f_k$。
    证明骨架: $g_k := \inf_{j\ge k} f_j$; (i) $0 \le g_k \le f_k$ ⟹ $g_k \in \IL$ (ii) $g_k$ 单调增 (iii) $\lim g_k = \liminf f_k$; $\int g_k \le \inf_{j\ge k}\int f_j$, 套递增 MCT。
  • Fatou a.e. 版: $f_k \ge 0$, $f_k \to f$ a.e. ⟹ $\int f \le \liminf\int f_k$ (衔接句: a.e. 收敛 ⟹ $\liminf f_k = f$ a.e.)
  • Reverse Fatou: 另设 $f_k \le F \in \IL$ ⟹ $\limsup\int f_k \le \int\limsup f_k$ (对 $F - f_k$ 用 Fatou, 消有限 $\int F$)
  • Gen. DCT: $|f_k| \le g_k$, $g_k \to g$ a.e., $\int g_k \to \int g < \infty$, $f_k \to f$ a.e. ⟹ $\int f_k \to \int f$ (Fatou 于 $g_k \pm f_k$ 双侧夹; "收敛项可提出 liminf")
  • Scheffé: $f_k \to f$ a.e. ⟹ [$\int|f_k| \to \int|f| \iff \int|f_k - f| \to 0$] (取 $h_k = |f_k - f| \le |f_k| + |f| =: g_k \to 2|f|$, 套 Gen. DCT)
  • 可积性白送模式: $f_k \ge 0$, $f_k \to f$ a.e., $\sup\int f_k = C < \infty$ ⟹ $f \in \IL$ 且 $\int f \le C$ (Fatou, 不需要 dominator!)
  • 桥定理 (4.11.19 题): 非负 $f$: $f \in \IL \iff$ 改进 Riemann 积分绝对收敛, 且数值相等。反例 $\frac{\sin x}{x}$: 条件收敛 (分部积分) 但 $\notin \IL$ ($|{\cdot}|$ 逐段 $\ge \frac{2}{(n+1)\pi}$, 调和发散)

决策树 + 答题模板 + 反例

选定理: 单调? → MCT | 固定可积 dominator? → DCT | 非负 + 单边不等式? → Fatou | dominator 在动? → Gen DCT | 问 $\int|f_k - f| \to 0$? → Scheffé

模板 (4–10 行): 点名定理 → 逐条验设 (单调性 / $\int F < \infty$ 算出来 / a.e. 极限求出来) → 套结论

  • Traveling box $\mathbf 1_{[k,k+1]}$: 逃 $\infty$; dominator $\ge \mathbf 1_{[1,\infty)}$ 不可积
  • Spike $k\mathbf 1_{(0,1/k]}$ / $\frac{k}{1+k^2x^2}\mathbf 1_{[0,1]}$ ($\int = \arctan k \to \frac\pi2$): 集中到点; dominator $\gtrsim \frac1x$ 不可积
  • 震荡箱 ($\mathbf 1_{[0,1]}$/$\mathbf 1_{[1,2]}$ 交替): liminf 逐点取最差子列, $\int\liminf = 0 < 1$ — 一致有界 + 公共支撑也救不了, 还缺逐点收敛
  • 负 spike $-k\mathbf 1_{(0,1/k]}$: 非负假设删掉 Fatou 反向

MATH 31CH Final — Side 2: 曲线曲率 · 曲面曲率 · Practice Final 答案精选

平面曲线 $c(t) = (x, y)$

  • $\kappa = \dfrac{x'y'' - y'x''}{((x')^2 + (y')^2)^{3/2}} = \dfrac{\det(\dot c, \ddot c)}{|\dot c|^3}$ (signed, $+$ = 向左弯)
  • 图象 $(t, f(t))$: $\kappa = \dfrac{f''}{(1+(f')^2)^{3/2}}$; 圆 (半径 $a$): $\kappa \equiv 1/a$

$(a\cos t, a\sin t)$$1/a$常数

空间曲线: 公式 + 流程

  • $\kappa = \dfrac{|\dot c \times \ddot c|}{|\dot c|^3} > 0$; $\;\tau = \dfrac{\det(\dot c, \ddot c, \dddot c)}{|\dot c \times \ddot c|^2}$ τ 分母 = 叉积平方, 不是 $|\dot c|^6$!
  • 流程: 算 $\dot c, \ddot c, \dddot c$ → $\dot c\times\ddot c$ 与 $|\dot c\times\ddot c|$ → $\kappa$ → $\det[\dot c|\ddot c|\dddot c]$ → $\tau$。det 技巧: 多项式曲线 $(t, t^2, t^3)$ 型的 $[\dot c|\ddot c|\dddot c]$ 是上三角, det = 对角积; 螺旋型沿"常数列"展开
  • 不需要先化 unit-speed (公式自带 $|\dot c|$); 直线 $\ddot c = 0$: Frenet 失效, $\tau$ 无定义
  • Frenet: $\T = \frac{\dot c}{|\dot c|}, \NN = \frac{\dot\T}{|\dot\T|}, \B = \T{\times}\NN$; unit-speed: $\T' = \kappa\NN$, $\NN' = -\kappa\T + \tau\B$, $\B' = -\tau\NN$ (OH: frame 理论大概率不考)

曲面六步流水线

  • ① $X_u, X_v$ → $\I = \begin{pmatrix} E & F \\ F & G \end{pmatrix}$, $E = X_u{\cdot}X_u$, $F = X_u{\cdot}X_v$, $G = X_v{\cdot}X_v$
  • ② $\nu = \frac{X_u\times X_v}{|X_u\times X_v|}$, $|X_u{\times}X_v| = \sqrt{EG-F^2}$; 面积 $= \iint\sqrt{EG-F^2}$ 不是 $\sqrt{EG}$
  • ③ $X_{uu}, X_{uv}, X_{vv}$ → $\II = \begin{pmatrix} L & M \\ M & N \end{pmatrix}$, $L = X_{uu}{\cdot}\nu$ 等
  • ④ $K = \dfrac{LN - M^2}{EG - F^2}$, $\;H = \dfrac{EN - 2FM + GL}{2(EG - F^2)}$ 分母有 2
  • ⑤ $S = \I^{-1}\II$, 主曲率 = 特征值; $K = k_1k_2$, $H = \frac{k_1+k_2}{2}$; 由 $H, K$ 反解: $k_{1,2} = H \pm \sqrt{H^2 - K}$
  • ⑥ $F = M = 0$ 时直接 $k_1 = L/E$, $k_2 = N/G$ (球/柱/旋转面/torus 全走这条)
  • $K$ 内蕴 (Theorema Egregium), 不依赖 $\nu$; $H$ 随 $\nu$ 翻转变号; $K{=}0$ ≠ 平面 (柱/锥 developable)

Graph $z = f(x,y)$, $W = \sqrt{1+f_x^2+f_y^2}$

  • $\I = \begin{pmatrix} 1{+}f_x^2 & f_xf_y \\ f_xf_y & 1{+}f_y^2 \end{pmatrix}$, $\nu = \frac{(-f_x,-f_y,1)}{W}$, $\II = \frac1W\begin{pmatrix} f_{xx} & f_{xy} \\ f_{xy} & f_{yy} \end{pmatrix}$
  • $K = \dfrac{f_{xx}f_{yy} - f_{xy}^2}{W^4}$, $\;H = \dfrac{(1{+}f_y^2)f_{xx} - 2f_xf_yf_{xy} + (1{+}f_x^2)f_{yy}}{2W^3}$
  • PF VII.4 椭圆抛物面 $z = \frac{x^2+y^2}{2}$ ($r^2 = x^2{+}y^2$, $w = 1{+}r^2$): $K = \frac{1}{w^2}$, $H = \frac{2+r^2}{2w^{3/2}}$, $k_1 = \frac{1}{\sqrt w}$, $k_2 = \frac{1}{w^{3/2}}$
  • 双曲抛物面 $z = \frac{x^2-y^2}{2}$: $K = \frac{-1}{w^2}$, $H = \frac{y^2-x^2}{2w^{3/2}}$; minimal points: $y = \pm x$

Torus (PF Part VIII, Final Review 压轴)

$X = ((R{+}r\cos v)\cos u, (R{+}r\cos v)\sin u, r\sin v)$, $R > r > 0$; $v{=}0$ 最外圈, $v{=}\pi$ 最内圈

  • $\I = \mathrm{diag}((R{+}r\cos v)^2, r^2)$; 外法向 $\nu = (\cos v\cos u, \cos v\sin u, \sin v)$; $\II = \mathrm{diag}(-(R{+}r\cos v)\cos v, -r)$
  • $k_1 = \frac{-\cos v}{R+r\cos v}$, $k_2 = -\frac1r$; $\;K = \dfrac{\cos v}{r(R+r\cos v)}$; $\;H = -\dfrac{R+2r\cos v}{2r(R+r\cos v)}$
  • $\mathrm{sgn}\,K = \mathrm{sgn}(\cos v)$: 外半圈 ($|v|{<}\frac\pi2$) $K>0$ / 内半圈 $K<0$ / 顶底圈 ($v = \pm\frac\pi2$) $K=0$
  • Minimal ($H{=}0 \iff \cos v = -\frac{R}{2r}$): $R > 2r$ 无 | $R = 2r$: $v{=}\pi$ 最内圈 | $r < R < 2r$: 两圆周 $v = \pm\arccos(-\frac{R}{2r})$

径向积分 & CoV Jacobian 工具箱

  • 极坐标 $|\det| = r$; 柱坐标 $= r$; 球坐标 (纬度 $\varphi$, $z = r\sin\varphi$) $= r^2\cos\varphi$
  • 径向降维: $\int_{\R^n} g(|\mathbf x|)\,dV = \sigma_n\int_0^\infty g(r)\,r^{n-1}dr$; $\sigma_1 = 2$, $\sigma_2 = 2\pi$, $\sigma_3 = 4\pi$, $\sigma_n = n\beta_n$ (单位球面积; 有限常数, 不影响收敛性判断)
  • $p$-阈值: $\int_{|\mathbf x| \ge 1}|\mathbf x|^p < \infty \iff p < -n$; $\int_{|\mathbf x| \le 1}|\mathbf x|^p < \infty \iff p > -n$ (球壳 $r^{n-1}$ 抬指数)
  • 1-D 判据: $\int_0^1 x^a < \infty \iff a > -1$; $\int_1^\infty x^a < \infty \iff a < -1$; $\int_0^1(-\ln x) = 1$
  • 夹逼: 证可积只要上界 $f \le Cg$; 证发散只要下界 $f \ge cg$; 双边夹 ⟹ 同收敛
  • 指数压多项式: $e^{t} \ge t^k/k!$ ⟹ $e^{-r^2} \le k!\,r^{-2k}$ ($k$ 任选) ⟹ $\int r^N e^{-r^2}dr < \infty$, Gaussian 矩全存在

考场提醒 (低级错误清单)

  • FT 约定: 本课正变换 $e^{+2\pi i\xi x}$ (与多数教材反!); 微分公式 $-2\pi i\xi$
  • $\LT(f')$ 漏 $-f(0)$ = 全错; ODE 答案代回初值验算 (10 秒保命)
  • $\tau$ 分母 $|\dot c\times\ddot c|^2$; 平面 $\kappa$ signed / 空间 $\kappa > 0$
  • $K$ 用 $W^4$, $H$ 用 $2W^3$ 且分子交叉配权 ($1{+}f_y^2$ 配 $f_{xx}$!); 面积元 $\sqrt{EG-F^2}$

  • DCT 的 dominator 可积性是要验的: 算出或界出 $\int F < \infty$
  • 球坐标 Jacobian $r^2\cos\varphi$ (纬度约定); 热核里 $a = \frac{1}{4t}$ 代 Gaussian 公式别代错
