MATH 31CH Final — Side 1: Fourier · Laplace · 收敛定理

Fourier 变换 (本课约定: 正变换 +2πi !)

$\FT{f}(\zeta) = \int_{\R^n} f(x)\,e^{+2\pi i \zeta\cdot x} dx$, $\;f(x) = \int \FT{f}(\zeta)\,e^{-2\pi i \zeta\cdot x} d\zeta$

  • 微分: $\widehat{\partial_{x_j} f} = -2\pi i \zeta_j \FT{f}$; $\;\widehat{\Delta f} = -4\pi^2|\zeta|^2 \FT{f}$
  • 卷积: $(f{*}g)(x) = \int f(x{-}y)g(y)dy$, $\;\widehat{f{*}g} = \FT{f}\,\FT{g}$; FT 在 $\IL$ 上单射
  • Gaussian: $\widehat{e^{-ax^2}} = \sqrt{\pi/a}\,e^{-\pi^2\zeta^2/a}$ ($n$D: $(\pi/a)^{n/2}e^{-\pi^2|\zeta|^2/a}$)
  • $\widehat{e^{-a|x|}} = \dfrac{2a}{a^2+4\pi^2\zeta^2}$; $\;\dfrac{1}{1+4\pi^2\zeta^2} = \tfrac12\widehat{e^{-|x|}}$
  • $\widehat{\mathbf 1_{[-A,A]}} = \dfrac{\sin(2\pi\zeta A)}{\pi\zeta}$ ($\zeta{=}0$: $2A$)

模板: 热方程 $u_t = \Delta u$, $u(0) = f$

FT$_x$ → $\partial_t\FT{u} = -4\pi^2|\zeta|^2\FT{u}$ → $\FT{u} = \FT{f}\,e^{-4\pi^2|\zeta|^2 t}$

热核 $K_t(x) = (4\pi t)^{-n/2} e^{-|x|^2/(4t)}$ ($a = \frac{1}{4t}$ 代 Gaussian), $\FT{K_t} = e^{-4\pi^2|\zeta|^2 t}$

$u = K_t * f$; $\;n{=}1$: $u(x,t) = \int_\R \frac{e^{-(x-y)^2/4t}}{\sqrt{4\pi t}} f(y)\,dy$

模板: Poisson 型 $-u'' + u = f$ (on $\R$)

$(1 + 4\pi^2\zeta^2)\FT{u} = \FT{f}$ → $\FT{u} = \frac{\FT{f}}{1+4\pi^2\zeta^2}$ → $u = \tfrac12 e^{-|\cdot|} * f$

($\R^3$ Yukawa $-\Delta u + m^2 u = f$: $G = e^{-m|x|}/(4\pi|x|)$)

Laplace 变换

$\LT(f)(s) = \int_0^\infty f(t)e^{-st}dt$

  • $\LT(f') = s\LT(f) - f(0)$; $\;\LT(f'') = s^2\LT(f) - sf(0) - f'(0)$ 初值勿漏!
  • $\LT(e^{-at}) = \frac{1}{s+a}$; $\;\LT(t^m e^{-at}) = \frac{m!}{(s+a)^{m+1}}$; $\;\LT(1) = \frac1s$; $\;\LT(t^m) = \frac{m!}{s^{m+1}}$
  • $\LT(\sin bt) = \frac{b}{s^2+b^2}$; $\;\LT(\cos bt) = \frac{s}{s^2+b^2}$
  • $\LT(t f(t)) = -\frac{d}{ds}\LT(f)$ ⟹ $\LT(t\sin t) = \frac{2s}{(s^2+1)^2}$, $\LT(t\cos t) = \frac{s^2-1}{(s^2+1)^2}$, $\frac{1}{(s^2+1)^2} = \LT\big(\tfrac12(\sin t - t\cos t)\big)$
  • 卷积: $(f{*}g)(t) = \int_0^t f(t{-}y)g(y)dy$, $\LT(f{*}g) = \LT(f)\LT(g)$

ODE 四步 + 两个范例

① 取 $\LT$ (带初值) ② 代数解出 $\LT(y)$ ③ 部分分式 ④ 查表逆变换; 末尾代回 $y(0)$ 验算

  • $y' + 3y = e^{-t}$, $y(0){=}2$: $\LT(y) = \frac{1/2}{s+1} + \frac{3/2}{s+3}$ → $y = \frac12 e^{-t} + \frac32 e^{-3t}$
  • $y'' + y = \sin t$, $y(0){=}0$, $y'(0){=}1$: $\LT(y) = \frac{1}{s^2+1} + \frac{1}{(s^2+1)^2}$ → $y = \frac32\sin t - \frac12 t\cos t$ (共振)

收敛定理 (Lebesgue, black box 引用)

  • MCT (4.11.18): $0 \le f_k \uparrow$ a.e., $\sup_k \int f_k < \infty$ ⟹ $f = \lim f_k \in \IL$, $\int f = \lim\int f_k$。递减版: $f_k \downarrow f$ a.e. + 某 $\int f_k$ 有限 (证: $f_1 - f_k$ 翻转)。
  • DCT (4.11.19): $f_k \to f$ a.e. + $|f_k| \le_L F \in \IL$ ⟹ $f \in \IL$, $\int|f_k - f| \to 0$, $\int f_k \to \int f$。必须验 $\int F < \infty$
  • Fatou (★ 教授 OH 钦点 "最有用"): $f_k \ge 0$ ⟹ $\int\liminf f_k \le \liminf\int f_k$ (证: $g_m = \inf_{k\ge m} f_k \uparrow$, $g_m \le f_m$, 套 MCT 后取 $\liminf$)。方向: 质量只漏不多。
  • Fatou a.e.: $f_k \ge 0$, $f_k \to f$ a.e. ⟹ $\int f \le \liminf\int f_k$
  • Gen. DCT: $|f_k| \le g_k$, $g_k \to g$ a.e., $\int g_k \to \int g$, $f_k \to f$ a.e. ⟹ $\int f_k \to \int f$ (证: Fatou 于 $g_k \pm f_k \ge 0$ 两侧夹)
  • Scheffé: $f_k \to f$ a.e. ⟹ [$\int|f_k| \to \int|f|$ ⟺ $\int|f_k - f| \to 0$] (证: $h_k = |f_k - f| \le g_k = |f_k| + |f|$ 套 Gen. DCT)

决策树 + 反例

单调? → MCT | 固定 dominator? → DCT | 非负 + 不等式? → Fatou | dominator 在动? → Gen. DCT | 问 $\int|f_k - f| \to 0$? → Scheffé

答题: 点名定理 → 逐条验设 (单调 / $\int F < \infty$ / a.e. 极限) → 套结论

  • Traveling box $\mathbf 1_{[k,k+1]}$: $\to 0$ 逐点, $\int \equiv 1$ (质量逃 $\infty$, dominator $\ge \mathbf 1_{[1,\infty)}$ 不可积)
  • Spike $k\mathbf 1_{(0,1/k]}$: $\to 0$ 逐点, $\int \equiv 1$ (质量集中 0, dominator $\ge 1/x$ 不可积)

MATH 31CH Final — Side 2: 曲线曲率 · 曲面曲率

平面曲线 $c(t) = (x, y)$

  • $\kappa = \dfrac{x'y'' - y'x''}{((x')^2 + (y')^2)^{3/2}} = \dfrac{\det(\dot c, \ddot c)}{|\dot c|^3}$ (signed; $+$ = 向左弯)
  • 图象 $(t, f(t))$: $\kappa = \dfrac{f''}{(1 + (f')^2)^{3/2}}$; 圆 (半径 $r$): $\kappa \equiv 1/r$

空间曲线 $c(t) \in \R^3$

  • $\kappa = \dfrac{|\dot c \times \ddot c|}{|\dot c|^3} > 0$; $\;\tau = \dfrac{\det(\dot c, \ddot c, \dddot c)}{|\dot c \times \ddot c|^2}$ τ 分母 = 叉积平方, 不是 $|\dot c|^6$!
  • Frenet: $\T = \frac{\dot c}{|\dot c|}$, $\NN = \frac{\dot\T}{|\dot\T|}$, $\B = \T \times \NN$; unit-speed 方程: $\T' = \kappa\NN$, $\NN' = -\kappa\T + \tau\B$, $\B' = -\tau\NN$
  • Helix $(a\cos t, a\sin t, bt)$: $\kappa = \frac{a}{a^2+b^2}$, $\tau = \frac{b}{a^2+b^2}$ (全常数)
  • 公式不需 unit-speed (已含 $|\dot c|$ 因子); 直线: Frenet 失效, $\tau$ 无定义

曲面流水线 $X(u,v): U \to \R^3$

  • ① $X_u, X_v$ → $\I = \begin{pmatrix} E & F \\ F & G \end{pmatrix}$, $E = X_u{\cdot}X_u$, $F = X_u{\cdot}X_v$, $G = X_v{\cdot}X_v$
  • ② $\nu = \frac{X_u \times X_v}{|X_u \times X_v|}$, $|X_u{\times}X_v| = \sqrt{EG - F^2}$; 面积 $= \iint\sqrt{EG-F^2}\,du\,dv$ 不是 $\sqrt{EG}$
  • ③ $X_{uu}, X_{uv}, X_{vv}$ → $\II = \begin{pmatrix} L & M \\ M & N \end{pmatrix}$, $L = X_{uu}{\cdot}\nu$, $M = X_{uv}{\cdot}\nu$, $N = X_{vv}{\cdot}\nu$
  • ④ $K = \dfrac{LN - M^2}{EG - F^2}$, $\;H = \dfrac{EN - 2FM + GL}{2(EG - F^2)}$ 分母有 2
  • ⑤ $S = \I^{-1}\II$; 主曲率 $k_1, k_2$ = 特征值; $K = k_1k_2$, $H = \frac{k_1+k_2}{2}$
  • ⑥ $F = M = 0$ 时直接 $k_1 = L/E$, $k_2 = N/G$
  • $K$ 内蕴 (Theorema Egregium), 不依赖 $\nu$; $H$ 随 $\nu$ 翻转变号

Graph $z = f(x,y)$, $\;W = \sqrt{1 + f_x^2 + f_y^2}$

  • $\I = \begin{pmatrix} 1{+}f_x^2 & f_xf_y \\ f_xf_y & 1{+}f_y^2 \end{pmatrix}$, $\;\nu = \frac{(-f_x, -f_y, 1)}{W}$, $\;\II = \frac1W\begin{pmatrix} f_{xx} & f_{xy} \\ f_{xy} & f_{yy} \end{pmatrix}$
  • $K = \dfrac{f_{xx}f_{yy} - f_{xy}^2}{W^4}$, $\;H = \dfrac{(1{+}f_y^2)f_{xx} - 2f_xf_yf_{xy} + (1{+}f_x^2)f_{yy}}{2W^3}$
  • 例: $z = \frac{x^2-y^2}{2}$: $K = \frac{-1}{(1+x^2+y^2)^2}$, $H = \frac{y^2 - x^2}{2(1+x^2+y^2)^{3/2}}$; minimal points: $y = \pm x$

Torus $X = ((R{+}r\cos v)\cos u, (R{+}r\cos v)\sin u, r\sin v)$, $R > r$

  • $\I = \mathrm{diag}((R{+}r\cos v)^2,\, r^2)$; 外法向 $\nu = (\cos v\cos u, \cos v\sin u, \sin v)$
  • $\II = \mathrm{diag}(-(R{+}r\cos v)\cos v,\, -r)$; $\;k_1 = \frac{-\cos v}{R + r\cos v}$, $k_2 = -\frac1r$
  • $K = \dfrac{\cos v}{r(R + r\cos v)}$: 外半圈 $>0$ / 内半圈 $<0$ / $v = \pm\frac\pi2$ 时 $=0$
  • $H = 0 \iff \cos v = -\frac{R}{2r}$: $R > 2r$ 无解 | $R = 2r$: $v = \pi$ 最内圈 | $r < R < 2r$: 两个圆周 $v = \pm\arccos(-\frac{R}{2r})$

标志曲面 (直接报数)

曲面$K$$H$
球面 $R$$1/R^2$$|H| = 1/R$
圆柱 $r$$0$$|H| = \frac{1}{2r}$
平面$0$$0$
鞍面 $z{=}uv$ @ $0$$-1$$0$ (minimal)

$K = 0$ ≠ 平面 (圆柱/圆锥 developable); minimal point: $H(p) = 0$; minimal surface: $H \equiv 0$

考场提醒

  • FT 约定: 正变换 $e^{+2\pi i\zeta x}$ (与 Stein/Folland 反!)
  • $\LT(f')$ 漏 $-f(0)$ = 全错; 答案代回初值验算
  • $K$ 用 $W^4$, $H$ 用 $2W^3$; 面积元 $\sqrt{EG - F^2}$
  • 平面 $\kappa$ signed / 空间 $\kappa > 0$; $\tau$ 分母叉积平方
  • DCT: dominator 可积性是要验的, 不是白送的
